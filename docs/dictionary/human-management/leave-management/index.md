# Gestión de Permisos

Gestión de permisos, licencias y vacaciones de empleados. Esta sección contiene las herramientas para la gestión de solicitudes de permiso y vacaciones del personal.

## Componentes

| Componente | Tipo | Descripción |
|-----------|------|-------------|
| Motivos de Permiso | Ventana | Define los motivos disponibles para solicitudes de permiso |
| Tipo de Permiso | Ventana | Clasifica los tipos de permisos disponibles (vacaciones, enfermedad, etc.) |
| Solicitud de Permiso | Ventana | Registra y gestiona las solicitudes de permiso de los empleados |
| Crear Permiso Repetido para Empleado | Proceso | Crea permisos recurrentes para un empleado basados en frecuencia y fecha de asignación |
