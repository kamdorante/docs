# ADM de Proveedor

## Descripción

Ventana para gestionar las Autorizaciones de Devolución de Material a proveedores. Permite registrar solicitudes de devolución al proveedor.

## Acceso

Menú: Gestión de Devoluciones → ADM de Proveedor
