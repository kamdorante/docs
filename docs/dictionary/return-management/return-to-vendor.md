# Devolución a Proveedor

## Descripción

Ventana para registrar las entregas de productos devueltos a proveedores. Se genera a partir de una ADM de proveedor aprobada y registra el despacho físico del material al proveedor.

## Acceso

Menú: Gestión de Devoluciones → Devolución a Proveedor

## Pestañas

### Entrega de Devolución

Encabezado del documento de entrega de devolución al proveedor, incluyendo datos del proveedor, almacén y fecha de despacho.

### Línea de Entrega

Líneas del documento con los productos devueltos, cantidades despachadas y su procedencia en el almacén.
