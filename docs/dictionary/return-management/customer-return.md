# Devolución de Cliente

## Descripción

Ventana para registrar las recepciones de productos devueltos por clientes. Se genera a partir de una ADM de cliente aprobada y registra el ingreso físico del material al almacén.

## Acceso

Menú: Gestión de Devoluciones → Devolución de Cliente

## Pestañas

### Recepción de Devolución

Encabezado del documento de recepción de la devolución de cliente, incluyendo datos del cliente, almacén y fecha de recepción.

### Línea de Recepción

Líneas del documento con los productos devueltos, cantidades recibidas y ubicación de almacenamiento.
