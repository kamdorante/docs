# ADM de Cliente

## Descripción

Ventana de transacción para gestionar las Autorizaciones de Devolución de Material de clientes (Customer RMA). Una ADM puede ser requerida para aceptar devoluciones y crear notas de crédito.

## Acceso

Menú: Gestión de Devoluciones → ADM de Cliente

## Pestañas

### ADM Cliente

Encabezado de la autorización de devolución de cliente. Define el socio de negocio, almacén, fecha y demás parámetros de la devolución.

### Línea ADM

Líneas individuales de la devolución con el detalle de los productos, cantidades y condiciones a devolver.
