# Tipo de ADM

## Descripción

Ventana para definir los tipos de Autorización de Devolución de Material (ADM). Permite configurar diferentes categorías de devolución para clasificar las ADM del sistema.

## Acceso

Menú: Gestión de Devoluciones → Tipo de ADM

## Pestañas

### Tipo ADM

Define los tipos de autorización de devolución disponibles en el sistema.
