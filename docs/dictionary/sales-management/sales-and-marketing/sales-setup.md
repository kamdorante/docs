# Configuración de Ventas

## Descripción
Flujo de trabajo para la configuración de representantes de ventas, comisiones, así como canales y campañas de mercadeo.

## Acceso
Menú: Gestión de Ventas → Ventas y Mercadeo → Configuración de Ventas

## Flujo de Trabajo
Este flujo de trabajo guía al usuario a través de los pasos necesarios para configurar el módulo de ventas y mercadeo del sistema:

1. **Región de Ventas** - Definir las regiones de cobertura de ventas
2. **Canal de Mercadeo** - Configurar los canales de mercadeo disponibles
3. **Campaña de Mercadeo** - Crear campañas de mercadeo
4. **Definición de Comisión** - Establecer las reglas de comisiones
5. **Ejecución de Comisión** - Procesar y revisar las comisiones
6. **Tipo de Comisión** - Definir tipos personalizados de comisión

## Notas
- Este flujo de trabajo es de tipo guía y no requiere procesamiento de documentos.
- Cada paso abre la ventana correspondiente para la configuración.
