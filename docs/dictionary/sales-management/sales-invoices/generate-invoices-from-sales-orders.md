# Generar Facturas desde Órdenes

## Descripción

Generar Facturas desde Órdenes de Venta seleccionadas. Este visor permite generar facturas o notas de crédito desde órdenes de venta seleccionadas dependiendo de la regla de facturación.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Generar Facturas desde Órdenes

## Columnas de resultado

| Campo | Descripción |
|---|---|
| Orden | Orden de venta asociada |
| Tipo de Documento | Tipo de documento o reglas |
| Socio del Negocio | Identifica al socio del negocio |
| No. del Documento | Número de secuencia del documento |
| Fecha de la Orden | Fecha en que se realizó la orden |
| Total de Líneas | Suma total de las líneas del documento |
| Gran Total | Monto total del documento incluyendo impuestos |
