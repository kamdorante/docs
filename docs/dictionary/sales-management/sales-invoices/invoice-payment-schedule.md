# Programa de Pagos de Facturas

## Descripción

Mantenga el programa de pagos de facturas sin pagar. Esta ventana permite consultar y gestionar los calendarios de pago asociados a las facturas pendientes de cobro.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Programa de Pagos de Facturas

## Pestañas

### Factura

Selección de facturas no pagadas completamente.

#### Campos principales

| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Orden | Orden de venta asociada | Texto | No |
| Tipo de Documento | Tipo de documento o reglas | Texto | No |
| No. del Documento | Número de secuencia del documento | Texto | No |
| Estado del Documento | Estado actual del documento | Texto | No |
| Fecha de Facturación | Fecha impresa en la factura | Texto | No |
| Agente Comercial | Representante de ventas asignado | Texto | No |
| Autoservicio | Indica si es una entrada de autoservicio | Texto | No |
| Descripción | Descripción corta opcional del registro | Texto | No |
| Socio del Negocio | Identifica al socio del negocio | Texto | No |
| Dirección del Socio | Dirección del socio del negocio | Texto | No |
| Usuario/Contacto | Contacto del socio del negocio | Texto | No |
| Moneda | Moneda del documento | Texto | No |
| Total de Líneas | Suma total de las líneas | Texto | No |
| Gran Total | Monto total del documento incluyendo impuestos | Texto | No |
| Pagado | Indica si la factura ha sido pagada | Texto | No |

### Programa de Pagos

Mantener el programa de pagos de las facturas.

#### Campos principales

| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Programa de Pago de Factura | Identificador del programa de pago | Texto | No |
| Procesado | Indica si el registro ha sido procesado | Texto | No |
| Factura | Factura asociada | Texto | No |
| Calendario de Pago | Calendario de pago definido | Texto | No |
| Fecha de Vencimiento | Fecha en que vence el pago | Texto | No |
| Monto Adeudado | Monto que se debe pagar | Texto | No |
| Fecha de Descuento | Fecha límite para obtener descuento por pronto pago | Texto | No |
| Monto del Descuento | Monto del descuento por pronto pago | Texto | No |
| Válido | Indica si el registro del programa de pago es válido | Texto | No |
