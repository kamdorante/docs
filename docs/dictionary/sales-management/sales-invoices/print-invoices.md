# Imprimir Facturas

## Descripción

Imprima las facturas en papel o para enviar en PDF. Permite (re)imprimir facturas o enviarlas como adjuntos PDF al contacto del socio del negocio que tenga una dirección de correo electrónico válida. Si solo ingresa un valor en el campo No. del Documento DESDE, se imprimirán todas las facturas con número igual o mayor; alternativamente puede utilizar el carácter % como comodín.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Imprimir Facturas

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Enviar PDF por Correo | Envía los documentos PDF por correo electrónico al socio del negocio | Sí/No | No | |
| Plantilla de Correo | Plantilla de texto para el envío por correo | Búsqueda directa | No | |
| Fecha de Facturación | Fecha impresa en la factura | Fecha | No | |
| Socio del Negocio | Identifica al socio del negocio | Búsqueda | No | |
| Factura | Factura específica a imprimir | Búsqueda | No | |
| No. del Documento | Número de secuencia del documento | Texto | No | |
