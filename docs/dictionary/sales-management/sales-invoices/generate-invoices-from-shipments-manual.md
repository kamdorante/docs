# Genera Facturas desde Embarques (Manual)

## Descripción

Permite generar facturas desde documentos de entrega de forma manual e interactiva. Este formulario proporciona una interfaz donde el usuario puede seleccionar las entregas completadas para generar las facturas correspondientes.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Genera Facturas desde Embarques (Manual)

## Funcionalidad

Este formulario permite al usuario revisar y seleccionar manualmente los documentos de entrega completados que serán utilizados para generar facturas. Proporciona mayor control sobre el proceso al permitir una selección individual de entregas, a diferencia del proceso automático que procesa todas las entregas pendientes según los criterios definidos.
