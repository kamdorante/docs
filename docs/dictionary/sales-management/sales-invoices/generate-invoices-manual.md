# Generar Facturas (Lote)

## Descripción

Generar e imprimir facturas manualmente. Este formulario permite al usuario seleccionar de forma interactiva las órdenes de venta para las cuales se generarán facturas, proporcionando mayor control sobre el proceso de facturación.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Generar Facturas (Lote)

## Funcionalidad

Este formulario proporciona una interfaz interactiva donde el usuario puede revisar las órdenes de venta pendientes de facturación y seleccionar cuáles incluir en el proceso de generación. A diferencia del proceso automático, permite una selección manual detallada de las órdenes a facturar, facilitando el control del proceso de facturación por lotes.
