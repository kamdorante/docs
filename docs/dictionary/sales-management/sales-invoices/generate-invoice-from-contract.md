# Generar Factura desde Concepto

## Descripción

Este visor inteligente permite generar facturas a partir de conceptos de informes de gasto vinculados a contratos de servicio. Facilita la facturación de servicios recurrentes asociados a contratos.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Generar Factura desde Concepto

## Columnas de resultado

| Campo | Descripción |
|---|---|
| Informes de Gasto a Facturar | Registro de informe de gasto pendiente de facturación |
| Socio de Facturación | Socio del negocio al que se facturará |
| Socio del Negocio | Identifica al socio del negocio |
| Fecha de Gasto | Fecha del gasto reportado |
| Producto | Producto o servicio asociado |
| Moneda | Moneda del monto |
| Monto del Gasto | Monto del gasto reportado |
| No. del Documento | Número de secuencia del documento |
| Cantidad | Cantidad del producto o servicio |
| Impuesto | Tipo de impuesto aplicado |
| Línea de Informe de Gasto | Línea del informe de gasto |
| Informe de Gasto | Informe de gasto asociado |
| Autoservicio | Indica si es entrada de autoservicio |
| Grupo de Socio del Negocio | Grupo al que pertenece el socio |
| Categoría del Producto | Categoría a la que pertenece el producto |
