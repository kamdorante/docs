# Generar Factura desde Cuotas de Orden

## Descripción

Este visor inteligente permite generar facturas a partir de cuotas definidas en los programas de facturación de las órdenes de venta. Facilita la facturación parcial y programada según los calendarios de facturación establecidos.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Generar Factura desde Cuotas de Orden

## Columnas de resultado

| Campo | Descripción |
|---|---|
| Cuota de Facturación de Orden | Identificador de la cuota de facturación |
| Orden | Orden de venta asociada |
| Socio del Negocio | Identifica al socio del negocio |
| Monto | Monto de la cuota |
| Fecha de Vencimiento | Fecha de vencimiento de la cuota |
| Número de Cuota | Número secuencial de la cuota |
| Descripción | Descripción de la cuota |
| Proyecto | Proyecto financiero asociado |
| Contrato | Contrato de servicio asociado |
| Agente Comercial | Representante de ventas asignado |
