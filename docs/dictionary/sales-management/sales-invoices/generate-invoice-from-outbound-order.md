# Generar Factura desde Orden de Salida

## Descripción

Esta búsqueda inteligente permite seleccionar las líneas de orden de salida listas para facturar. Facilita la generación de facturas directamente desde las órdenes de salida del almacén.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Generar Factura desde Orden de Salida

## Columnas de resultado

| Campo | Descripción |
|---|---|
| Línea de Orden de Salida | Línea de la orden de salida |
| Orden de Salida | Orden de entrada y salida del almacén |
| Orden | Número de documento de la orden asociada |
| Producto | Producto, servicio o artículo |
| Cantidad Recogida | Cantidad preparada para envío |
| Cantidad del Movimiento | Cantidad del movimiento de inventario |
| Fecha de Envío | Fecha del envío |
| Nombre | Nombre del registro |
| Monto | Monto de la línea |
| Gran Total | Monto total del documento |
