# Generar Facturas desde Líneas de Orden con Agrupación

## Descripción

Este visor inteligente permite generar facturas seleccionando líneas de órdenes de venta con la capacidad de agrupar múltiples líneas en una sola factura. Proporciona información detallada de cantidades y permite consolidar la facturación.

## Acceso

Menú: Gestión de Ventas → Facturas de Venta → Generar Facturas desde Líneas de Orden con Agrupación

## Columnas de resultado

|---|---|
