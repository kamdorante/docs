# Navegador de Líneas de Proyectos

## Descripción

Navegador inteligente que permite consultar y gestionar las líneas de proyectos del sistema. Facilita la visualización detallada de los elementos de trabajo de múltiples proyectos en una sola vista.

## Acceso

Menú: Gestión de Ventas → Proyectos → Navegador de Líneas de Proyectos

## Campos del Browser

| Campo | Descripción | Filtro | Editable |
|-------|-------------|--------|----------|
| Organización | Organización de la línea | Sí | No |
| Proyecto | Proyecto al que pertenece la línea | Sí | No |
| Fase del Proyecto | Fase asociada a la línea | Sí | No |
| Tarea del Proyecto | Tarea asociada a la línea | Sí | No |
| Producto | Producto o servicio de la línea | Sí | No |
| Descripción | Descripción de la línea | No | No |
| Cantidad Planificada | Cantidad planificada para la línea | No | No |
| Precio Planificado | Precio planificado unitario | No | No |
| Monto Planificado | Monto total planificado | No | No |
| Cantidad Entregada | Cantidad realmente entregada | No | No |
| Cantidad Facturada | Cantidad facturada al cliente | No | No |
| Monto Facturado | Monto total facturado | No | No |
