# Tipo de Miembro de Proyecto

## Descripción

Permite definir y mantener los tipos de miembro de un proyecto. El tipo de miembro de proyecto puede ser asignado a los miembros del equipo de proyecto para clasificar su rol dentro del mismo (por ejemplo: Gerente, Desarrollador, Analista, etc.).

## Acceso

Menú: Gestión de Ventas → Proyectos → Tipo de Miembro de Proyecto

## Pestañas

### Tipo de Miembro de Proyecto

#### Campos

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Clave de Búsqueda | Clave de búsqueda única | Texto | Sí |
| Nombre | Nombre del tipo de miembro | Texto | Sí |
| Descripción | Descripción del tipo de miembro | Texto | No |
| Comentario/Ayuda | Comentario o ayuda sobre el uso | Texto | No |
| Por Defecto | Indica si es el tipo de miembro por defecto | Sí/No | No |
