# Detalle de Avance de Proyecto

## Descripción

Reporte que muestra el detalle del avance de un proyecto. Permite visualizar el progreso por fases y tareas, mostrando el porcentaje de completitud y el estado actual de cada componente del proyecto.

## Acceso

Menú: Gestión de Ventas → Proyectos → Detalle de Avance de Proyecto

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Proyecto | Proyecto a reportar | Búsqueda | No |
| Fase del Proyecto | Fase específica del proyecto a reportar | Búsqueda | No |
