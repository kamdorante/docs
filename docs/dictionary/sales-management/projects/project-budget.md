# Presupuesto de Proyecto

## Descripción

Ventana para gestionar el presupuesto asociado a los proyectos. Permite definir y controlar los montos presupuestados para cada proyecto, facilitando el seguimiento financiero y la comparación entre lo planificado y lo ejecutado.

## Acceso

Menú: Gestión de Ventas → Proyectos → Presupuesto de Proyecto

## Pestañas

### Presupuesto de Proyecto

Información del presupuesto asignado al proyecto.

#### Campos

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Proyecto | Proyecto al que pertenece el presupuesto | Búsqueda | Sí |
| Nombre | Nombre o descripción del presupuesto | Texto | Sí |
| Descripción | Descripción detallada del presupuesto | Texto | No |
| Moneda | Moneda en la que se expresa el presupuesto | Tabla | No |
| Monto Total | Monto total presupuestado | Monto | No |
| Fecha de Inicio | Fecha de inicio del período presupuestado | Fecha | No |
| Fecha de Fin | Fecha de fin del período presupuestado | Fecha | No |
