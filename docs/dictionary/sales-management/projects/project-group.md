# Grupo de Proyecto

## Descripción

Permite definir y mantener grupos de proyectos para organizar y clasificar los proyectos de la organización. Los grupos facilitan la segmentación y el reporte de proyectos relacionados.

## Acceso

Menú: Gestión de Ventas → Proyectos → Grupo de Proyecto

## Pestañas

### Grupo de Proyecto

#### Campos

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Clave de Búsqueda | Clave de búsqueda única | Texto | Sí |
| Nombre | Nombre del grupo de proyecto | Texto | Sí |
| Descripción | Descripción opcional del grupo | Texto | No |
| Por Defecto | Indica si este grupo es el valor por defecto | Sí/No | No |
| Color de Impresión | Color utilizado para impresión y visualización | Tabla | No |
| Grupo de Proyecto Padre | Grupo padre para estructura jerárquica | Tabla Directa | No |
