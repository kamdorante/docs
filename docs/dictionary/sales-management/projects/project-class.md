# Clase de Proyecto

## Descripción

Permite definir y mantener clases de proyectos para una clasificación adicional dentro del sistema. Las clases de proyecto complementan la categorización de los proyectos junto con el grupo y la categoría.

## Acceso

Menú: Gestión de Ventas → Proyectos → Clase de Proyecto

## Pestañas

### Clase de Proyecto

#### Campos

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Clave de Búsqueda | Clave de búsqueda única | Texto | Sí |
| Nombre | Nombre de la clase de proyecto | Texto | Sí |
| Descripción | Descripción opcional de la clase | Texto | No |
| Por Defecto | Indica si esta clase es el valor por defecto | Sí/No | No |
| Color de Impresión | Color utilizado para impresión y visualización | Tabla | No |
| Clase de Proyecto Padre | Clase padre para estructura jerárquica | Tabla Directa | No |
