# Navegador de Proyectos

## Descripción

Navegador inteligente que permite consultar y visualizar los proyectos del sistema con sus datos clave. Facilita la búsqueda, filtrado y gestión masiva de proyectos desde una interfaz tabular.

## Acceso

Menú: Gestión de Ventas → Proyectos → Navegador de Proyectos

## Campos del Browser

| Campo | Descripción | Filtro | Editable |
|-------|-------------|--------|----------|
| Organización | Organización del proyecto | Sí | No |
| Clave de Búsqueda | Clave única del proyecto | Sí | No |
| Nombre | Nombre del proyecto | Sí | No |
| Socio de Negocio | Cliente asociado al proyecto | Sí | No |
| Tipo de Proyecto | Tipo del proyecto | Sí | No |
| Estado | Estado actual del proyecto | Sí | No |
| Fecha de Inicio | Fecha de inicio programada | Sí | No |
| Fecha de Fin | Fecha de finalización programada | Sí | No |
| Monto del Contrato | Monto total del contrato | No | No |
| Monto Planificado | Monto planificado | No | No |
| Gerente del Proyecto | Responsable del proyecto | Sí | No |
| % Completado | Porcentaje de avance | No | No |
