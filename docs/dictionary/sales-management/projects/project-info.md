# Información del Proyecto

## Descripción

Ventana de solo lectura que muestra información del documento del proyecto. Permite visualizar el estado consolidado de un proyecto incluyendo sus documentos asociados, sin posibilidad de modificación directa.

## Acceso

Menú: Gestión de Ventas → Proyectos → Información del Proyecto

## Pestañas

### Información del Proyecto

Vista consolidada con la información del proyecto y sus documentos relacionados.

#### Campos

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Proyecto | Proyecto a consultar | Búsqueda | Sí |
| Nombre | Nombre del proyecto | Texto | No |
| Descripción | Descripción del proyecto | Texto | No |
| Estado | Estado actual del proyecto | Tabla Directa | No |
| Socio de Negocio | Cliente asociado al proyecto | Búsqueda | No |
| Fecha de Inicio | Fecha de inicio programada | Fecha | No |
| Fecha de Fin | Fecha de finalización programada | Fecha | No |
| Monto del Contrato | Monto total del contrato | Monto | No |
| Monto Facturado | Total facturado al cliente | Monto | No |
| % Completado | Porcentaje de avance del proyecto | Número | No |
