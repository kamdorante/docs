# Entregas por Grupo de Producto

## Descripción
Reporte de entregas agrupadas por grupo de producto. Permite visualizar las entregas realizadas filtrando por período, lista de usuario y socio del negocio.

## Acceso
Menú: Gestión de Ventas → Consulta de Facturas → Entregas por Grupo de Producto

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Período | Período del calendario | Búsqueda | No | |
| Lista de Usuario 4 | Elemento de lista definido por el usuario #4 | Búsqueda | No | |
| Socio del Negocio | Identifica un Socio del Negocio | Búsqueda | No | |
