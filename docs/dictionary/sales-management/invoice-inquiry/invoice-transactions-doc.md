# Transacciones de Factura (Doc.)

## Descripción
El reporte lista las transacciones de factura por fecha de factura.

## Acceso
Menú: Gestión de Ventas → Consulta de Facturas → Transacciones de Factura (Doc.)

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Organización | Entidad organizacional dentro de la compañía | Búsqueda | No | Todas |
| Grupo de Socio del Negocio | Grupo del Socio del Negocio | Búsqueda | No | |
| Socio del Negocio | Identifica un Socio del Negocio | Búsqueda | No | |
| Transacción de Venta | Indica si es una transacción de venta | Lista | No | Sí |
| Tipo de Documento Destino | Tipo de documento destino para conversión de documentos | Búsqueda | No | |
| Fecha de Factura | Fecha impresa en la factura | Fecha | No | Hoy |
| Fecha Contable | Fecha contable del documento | Fecha | No | |
| Fecha de la Orden | Fecha de la orden | Fecha | No | |
| Estado del Documento | Estado actual del documento | Lista | No | |
| Orden | Documento de la orden | Búsqueda | No | |
| Factura | Documento de la factura | Búsqueda | No | |
| Regla de Pago | Método de pago de la factura | Lista | No | |
| Término de Pago | Condiciones de pago (plazos, descuentos) | Búsqueda | No | |
| Proyecto | Proyecto financiero | Búsqueda | No | |

| Agente Comercial | Representante de ventas o agente de la compañía | Búsqueda | No | |
