# Generar Entregas (Manual)

## Descripción

Permite generar e imprimir entregas de forma manual. Las entregas para órdenes abiertas se crean basándose en la regla de entrega de la orden y la prioridad relativa. Si se selecciona una fecha prometida, solo se incluyen las órdenes hasta esa fecha (inclusive). Si varias órdenes de un socio del negocio tienen la misma ubicación, las órdenes pueden ser consolidadas en una sola entrega.

## Acceso

Menú: Gestión de Ventas → Entregas → Generar Entregas (Manual)

## Funcionalidad

Este formulario permite al usuario seleccionar manualmente las órdenes de venta para las cuales se generarán entregas. A diferencia del proceso automático, proporciona una interfaz interactiva donde el usuario puede revisar y seleccionar las órdenes antes de generar los documentos de entrega.
