# Generar Entregas desde Línea de Orden

## Descripción

Este visor inteligente permite generar entregas seleccionando líneas de órdenes de venta pendientes de entrega. Excluye las líneas que ya tienen entregas en estado borrador, en proceso o incompletas.

## Acceso

Menú: Gestión de Ventas → Entregas → Generar Entregas desde Línea de Orden

## Columnas de resultado

| Campo | Descripción |
|---|---|
| Línea de Orden de Venta | Línea de la orden de venta |
| No. del Documento | Número de secuencia del documento |
| Socio del Negocio | Identifica al socio del negocio |
| Producto | Producto, servicio o artículo |
| Cantidad a Entregar | Cantidad pendiente de entrega |
| Cantidad Disponible | Cantidad disponible (disponible - reservada) |
| Cantidad Ordenada | Cantidad ordenada en la orden |
| Fecha de la Orden | Fecha en que se realizó la orden |
| Fecha Prometida | Fecha prometida de entrega |
| Tipo de Documento | Tipo de documento o reglas |
