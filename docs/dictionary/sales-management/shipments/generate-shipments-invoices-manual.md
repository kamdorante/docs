# Generar Entregas y Facturas (Manual)

## Descripción

Permite generar entregas y facturas de forma simultánea desde órdenes de venta abiertas. Este formulario combina el proceso de generación de entregas y facturación en un solo paso, agilizando el flujo de trabajo cuando ambos documentos deben crearse al mismo tiempo.

## Acceso

Menú: Gestión de Ventas → Entregas → Generar Entregas y Facturas (Manual)

## Funcionalidad

Este formulario permite al usuario seleccionar órdenes de venta pendientes y generar en una sola operación tanto el documento de entrega como la factura correspondiente. Es especialmente útil cuando la regla de facturación está configurada para facturar al momento de la entrega, eliminando la necesidad de ejecutar dos procesos separados.
