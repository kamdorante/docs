# Entregas

Esta sección contiene la documentación del diccionario de datos para los componentes del submódulo de Entregas dentro de la Gestión de Ventas.

## Ventanas

- [Entregas (Cliente)](shipment-customer.md): Define las entregas hechas o por hacer a un cliente. Las entregas son generadas desde una orden de un cliente.

## Visores Inteligentes

- [Generar Entregas desde Línea de Orden](generate-shipment-from-order-lines.md): Permite generar entregas seleccionando líneas de órdenes de venta pendientes de entrega.

## Formularios

- [Generar Entregas (Manual)](generate-shipments-manual.md): Permite generar e imprimir entregas de forma manual desde órdenes de venta abiertas.
- [Generar Entregas y Facturas (Manual)](generate-shipments-invoices-manual.md): Permite generar entregas y facturas de forma simultánea desde órdenes de venta abiertas.

## Reportes

- [Detalle de Entregas](shipment-details.md): Información detallada del encabezado y líneas de entregas de material con información de ubicación.
