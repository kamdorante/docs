# Detalle de Entregas

## Descripción

El informe proporciona información detallada del encabezado y líneas de entregas de material con información de ubicación. Puede utilizar el informe para crear una lista de recogida consolidada.

## Acceso

Menú: Gestión de Ventas → Entregas → Detalle de Entregas
