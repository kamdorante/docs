# Prospecto

## Descripción
La ventana de Prospecto permite gestionar los prospectos de venta. Estos son contactos no calificados a los que se desea realizar acciones de mercadeo. Un prospecto puede ser convertido en un socio del negocio cuando se califica.

## Acceso
Menú: Gestión de Ventas → Gestión de Ventas CRM → Prospecto

## Pestañas

### Prospecto
Información principal del prospecto incluyendo datos de contacto y estado.

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Nombre | Nombre del prospecto | Texto | Sí |
| Descripción | Descripción corta opcional | Texto | No |
| Socio del Negocio | Socio del Negocio asociado | Búsqueda | No |
| Es Prospecto | Indica que el contacto es un prospecto de ventas | Sí/No | Sí |
| Fuente del Prospecto | Fuente inicial del prospecto | Lista | No |
| Estado del Prospecto | Estado del prospecto en el pipeline | Lista | No |
| Fecha de Cierre Estimada | Fecha estimada de cierre | Fecha | No |
| Monto Estimado | Monto estimado de la oportunidad | Monto | No |
| Agente Comercial | Representante de ventas asignado | Búsqueda | No |
| Correo Electrónico | Dirección de correo electrónico | Texto | No |
| Teléfono | Número de teléfono | Texto | No |
| Campaña | Campaña de mercadeo asociada | Búsqueda | No |

### Actividad
Registro de actividades realizadas con el prospecto.

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Tipo de Actividad | Tipo de actividad realizada | Lista | Sí |
| Descripción | Descripción de la actividad | Texto | No |
| Fecha de Inicio | Fecha de inicio de la actividad | Fecha y hora | Sí |
| Fecha de Finalización | Fecha de finalización de la actividad | Fecha y hora | No |
| Completo | Indica si la actividad está completa | Sí/No | No |
