# Generar OC para Instalación de Producto

## Descripción
Consulta inteligente que permite generar órdenes de compra para la instalación de productos. Permite seleccionar las líneas de orden de venta que requieren instalación y generar las correspondientes órdenes de compra a los proveedores de servicio de instalación.

## Acceso
Menú: Gestión de Ventas → Gestión de Ventas CRM → Generar OC para Instalación de Producto

## Criterios de búsqueda
| Campo | Descripción | Tipo | Rango |
|---|---|---|---|
| Socio del Negocio | Socio del negocio asociado | Búsqueda | No |
| Orden | Orden de venta de origen | Búsqueda | No |
| Producto | Producto que requiere instalación | Búsqueda | No |

## Columnas de resultado
| Campo | Descripción |
|---|---|
| Socio del Negocio | Socio del Negocio de la orden |
| Orden | Orden de venta de referencia |
| Producto | Producto a instalar |
| Cantidad Consignada | Cantidad enviada en consignación |
| Cantidad Pendiente | Cantidad pendiente de instalación |
| Cantidad Vendida | Cantidad vendida |
| Cantidad a Facturar | Cantidad a incluir en la orden de compra |
