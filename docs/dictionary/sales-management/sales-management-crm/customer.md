# Cliente

## Descripción
La ventana de Cliente proporciona una vista CRM consolidada de la información del cliente, incluyendo oportunidades, contactos, actividades, ubicaciones, precios de contrato, solicitudes y órdenes.

## Acceso
Menú: Gestión de Ventas → Gestión de Ventas CRM → Cliente

## Pestañas

### Cliente
Información principal del cliente (Socio del Negocio marcado como cliente).

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Clave de Búsqueda | Clave de búsqueda del registro | Texto | Sí |
| Nombre | Nombre del cliente | Texto | Sí |
| Descripción | Descripción corta opcional | Texto | No |
| Grupo de Socio del Negocio | Grupo al que pertenece el cliente | Búsqueda | Sí |
| Agente Comercial | Representante de ventas asignado | Búsqueda | No |
| Lista de Precios | Lista de precios del cliente | Búsqueda | No |
| Término de Pago | Condiciones de pago | Búsqueda | No |
| Regla de Pago | Método de pago | Lista | No |
| Límite de Crédito | Monto máximo de crédito | Monto | No |

### Oportunidades
Oportunidades de venta asociadas al cliente.

### Contactos
Contactos del cliente.

### Actividades
Actividades realizadas con los contactos del cliente.

### Ubicaciones
Direcciones y ubicaciones del cliente.

### Precios de Contrato
Precios específicos del cliente.

### Solicitudes
Solicitudes y tickets asociados al cliente.

### Órdenes
Detalle de órdenes del cliente.
