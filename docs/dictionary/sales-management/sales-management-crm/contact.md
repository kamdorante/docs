# Contacto

## Descripción
La ventana de Contacto permite gestionar los contactos, que son individuos con los que se interactúa. Los contactos pueden ser usuarios internos o externos que pueden iniciar sesión en el sistema y acceder a funcionalidades a través de uno o más roles. Un contacto también puede ser un contacto de un socio del negocio.

## Acceso
Menú: Gestión de Ventas → Gestión de Ventas CRM → Contacto

## Pestañas

### Contacto
Información detallada del contacto.

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Nombre | Nombre del contacto | Texto | Sí |
| Descripción | Descripción corta opcional | Texto | No |
| Socio del Negocio | Socio del Negocio al que pertenece el contacto | Búsqueda | No |
| Correo Electrónico | Dirección de correo electrónico | Texto | No |
| Teléfono | Número de teléfono | Texto | No |
| Teléfono Móvil | Número de teléfono móvil | Texto | No |
| Fax | Número de fax | Texto | No |
| Cargo | Cargo o posición del contacto | Texto | No |
| Título | Título del contacto (Sr., Sra., etc.) | Texto | No |
| Cumpleaños | Fecha de cumpleaños del contacto | Fecha | No |
| Comentarios | Comentarios adicionales | Texto | No |

### Actividad
Registro de actividades realizadas con el contacto.

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Tipo de Actividad | Tipo de actividad realizada | Lista | Sí |
| Descripción | Descripción de la actividad | Texto | No |
| Fecha de Inicio | Fecha de inicio de la actividad | Fecha y hora | Sí |
| Fecha de Finalización | Fecha de finalización | Fecha y hora | No |
| Completo | Indica si la actividad está completa | Sí/No | No |
