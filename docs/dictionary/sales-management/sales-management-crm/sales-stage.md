# Etapa de Venta

## Descripción
La ventana de Etapa de Venta permite definir las diferentes etapas del pipeline de ventas. Cada etapa tiene una probabilidad asociada que se utiliza para calcular el valor ponderado de las oportunidades.

## Acceso
Menú: Gestión de Ventas → Gestión de Ventas CRM → Etapa de Venta

## Pestañas

### Etapa de Venta
Configuración de las etapas de venta.

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Clave de Búsqueda | Clave de búsqueda del registro | Texto | Sí |
| Nombre | Nombre de la etapa de venta | Texto | Sí |
| Descripción | Descripción de la etapa | Texto | No |
| Probabilidad | Probabilidad de cierre asociada a la etapa (%) | Numérico | No |
| Secuencia | Secuencia de la etapa en el pipeline | Numérico entero | No |
| Cerrado | Indica si la etapa representa un estado cerrado | Sí/No | No |
| Ganado | Indica si la etapa representa una venta ganada | Sí/No | No |

### Oportunidad
Oportunidades de venta que se encuentran en esta etapa.
