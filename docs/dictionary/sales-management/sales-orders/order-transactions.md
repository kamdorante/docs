# Transacciones de Órdenes

## Descripción

Reporte de transacciones de órdenes de venta que permite visualizar el detalle de todas las transacciones realizadas en un período determinado. Muestra información consolidada sobre las órdenes procesadas en el sistema.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Transacciones de Órdenes

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Fecha de la Orden | Rango de fechas para filtrar las transacciones | Fecha | No | Fecha actual |
| Transacción de Ventas | Indica si se filtran transacciones de venta (Sí) o de compra (No) | Sí/No | No | Sí |
