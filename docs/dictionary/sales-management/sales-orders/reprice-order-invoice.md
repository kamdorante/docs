# Recalcular Precio de Orden/Factura

## Descripción

Este proceso permite recalcular los precios de una orden o factura abierta basándose en la versión más reciente de la lista de precios. Es útil cuando se han actualizado las listas de precios y se desea aplicar los nuevos precios a documentos existentes que aún no han sido procesados.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Recalcular Precio de Orden/Factura

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Orden | Orden de venta abierta a la que se le recalcularán los precios | Búsqueda directa | No | |
| Factura | Factura abierta a la que se le recalcularán los precios | Búsqueda directa | No | |
