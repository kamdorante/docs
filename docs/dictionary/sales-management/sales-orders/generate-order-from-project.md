# Generar Orden desde Proyecto

## Descripción

Consulta inteligente que permite generar órdenes de venta o compra a partir de proyectos existentes en el sistema. Muestra los proyectos disponibles con sus detalles y permite seleccionar aquellos para los cuales se desea generar las órdenes correspondientes.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Generar Orden desde Proyecto

## Criterios de búsqueda

| Campo | Descripción | Tipo | Rango |
|---|---|---|---|
| Socio del Negocio | Cliente asociado al proyecto | Búsqueda | No |
| Proyecto | Proyecto específico | Búsqueda | No |
| Clave de Búsqueda | Clave de búsqueda del proyecto | Texto | No |
| Dirección del Socio | Dirección del socio del negocio | Búsqueda directa | No |
| Categoría de Proyecto | Categoría del proyecto | Búsqueda directa | No |
| Inicio Programado | Fecha de inicio programado del proyecto | Fecha | No |
| Producto | Producto asociado al proyecto | Búsqueda directa | No |
| Agente Comercial | Representante de ventas | Tabla | No |
| Contrato de Servicio | Contrato de servicio asociado | Búsqueda directa | No |
| Región de Ventas | Región de ventas del proyecto | Búsqueda directa | No |
