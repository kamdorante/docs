# Tipo de Suscripción

## Descripción

La ventana de Tipo de Suscripción permite mantener los diferentes tipos de suscripción disponibles en el sistema, definiendo la frecuencia de renovación para cada tipo.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Tipo de Suscripción

## Pestañas

### Tipo de Suscripción

Pestaña principal para definir los tipos de suscripción y su frecuencia de renovación.

#### Campos principales

| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Nombre | Identificador alfanumérico de la entidad | Texto | Sí |
| Descripción | Descripción corta opcional del registro | Texto | No |
| Tipo de Frecuencia | Tipo de frecuencia del evento (diaria, semanal, mensual, etc.) | Lista | Sí |
| Frecuencia | Frecuencia de los eventos. Se usa en conjunto con el tipo de frecuencia para determinar el período. Por ejemplo: si el tipo es Semana y la frecuencia es 2, será cada dos semanas | Numérico entero | Sí |
