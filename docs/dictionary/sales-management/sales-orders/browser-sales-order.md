# Consulta de Órdenes de Venta

## Descripción

Consulta inteligente que permite buscar y visualizar órdenes de venta con múltiples criterios de filtrado. Proporciona una vista completa de las órdenes del sistema con información detallada sobre socios del negocio, montos, estados y condiciones de entrega y facturación.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Consulta de Órdenes de Venta

## Criterios de búsqueda

| Campo | Descripción | Tipo | Rango |
|---|---|---|---|
| Orden | Identificador de la orden | Búsqueda directa | No |
| Almacén | Almacén de la orden | Búsqueda directa | No |
| Tipo de Documento | Tipo de documento | Búsqueda directa | No |
| No. del Documento | Número del documento | Texto | No |
| Referencia de Orden | Referencia externa de la orden | Texto | No |
| Socio del Negocio | Cliente de la orden | Búsqueda | No |
| Estado de Crédito | Estado de crédito del socio | Lista | No |
| Agente Comercial | Representante de ventas | Tabla | No |
| Fecha de la Orden | Fecha de la orden | Fecha | No |
| Fecha Prometida | Fecha prometida de entrega | Fecha | No |
| Estado del Documento | Estado actual del documento | Lista | No |
