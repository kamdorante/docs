# Reabrir Orden

## Descripción

Este proceso permite reabrir una orden que ha sido previamente cerrada. Se utiliza cuando es necesario realizar modificaciones o agregar líneas a una orden que fue cerrada anticipadamente.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Reabrir Orden

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Orden | Orden cerrada que se desea reabrir | Búsqueda | No | |
