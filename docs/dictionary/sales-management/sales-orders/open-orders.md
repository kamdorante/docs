# Órdenes Abiertas

## Descripción

Reporte de órdenes abiertas que muestra las órdenes con cantidades pendientes de entregar (pendientes de despacho) o cantidades pendientes de facturar. Permite identificar rápidamente el estado de cumplimiento de las órdenes activas.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Órdenes Abiertas

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Socio del Negocio | Filtrar por socio del negocio específico | Búsqueda | No | |
| Producto | Filtrar por producto específico | Búsqueda | No | |
| Transacción de Ventas | Indica si se filtran transacciones de venta (Sí) o de compra (No) | Sí/No | No | Sí |
