# Detalle de Órdenes

## Descripción

Reporte que muestra información detallada de las órdenes, incluyendo ventas perdidas. El reporte puede ser utilizado tanto para el área de ventas como de compras, proporcionando visibilidad completa sobre las líneas de las órdenes y sus estados.

## Acceso

Menú: Gestión de Ventas → Órdenes de Venta → Detalle de Órdenes
