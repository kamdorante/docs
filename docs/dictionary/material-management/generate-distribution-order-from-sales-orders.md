# Generar Orden de Distribucion desde Ordenes de Venta

## Descripcion

Genera una Orden de Distribucion desde las Ordenes de Ventas seleccionadas con almacen de referencia. Use este visor para seleccionar todas las ordenes de ventas que pueden ser agregadas a un reabastecimiento.

## Acceso

Menu: Gestion de Materiales → Generar Orden de Distribucion desde Ordenes de Venta

## Criterios de busqueda

| Campo | Descripcion | Tipo | Rango |
|---|---|---|---|
| Orden de Venta | Orden de venta asociada | Busqueda directa | No |
| Almacen | Almacen de almacenamiento | Busqueda directa | No |
| Producto | Producto, servicio o articulo | Busqueda | No |
| Fecha de la Orden | Fecha de la orden de venta | Fecha | Si |

## Columnas de resultado

| Campo | Descripcion |
|---|---|
| No. del Documento | Numero del documento de la orden |
| Socio del Negocio | Cliente de la orden |
| Producto | Producto de la linea |
| Cantidad | Cantidad ordenada |
| Almacen | Almacen de origen |
