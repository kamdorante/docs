# Reabastecimiento de Productos

## Descripcion

Reabastece productos que se encuentran pendientes por reponer. Use este visor para seleccionar los productos a reponer en funcion de las reglas seleccionadas.

## Acceso

Menu: Gestion de Materiales → Reabastecimiento de Productos

## Criterios de busqueda

| Campo | Descripcion | Tipo | Rango |
|---|---|---|---|
| Almacen | Almacen de almacenamiento | Busqueda directa | No |
| Producto | Producto, servicio o articulo | Busqueda | No |
| Categoria de Producto | Categoria del producto | Busqueda directa | No |

## Columnas de resultado

| Campo | Descripcion |
|---|---|
| Producto | Producto a reponer |
| Almacen | Almacen destino |
| Cantidad Disponible | Cantidad actual en inventario |
| Nivel Minimo | Nivel minimo de stock |
| Nivel Maximo | Nivel maximo de stock |
| Cantidad a Reponer | Cantidad sugerida para reposicion |
