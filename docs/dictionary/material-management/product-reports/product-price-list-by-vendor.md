# Lista de Precios por Proveedor

## Descripcion

Reporte de Lista de Precios por Proveedor. Permite visualizar todos los productos y sus precios agrupados por proveedor.

## Acceso

Menu: Gestion de Materiales → Reportes de Productos → Lista de Precios por Proveedor

## Parametros

| Parametro | Descripcion | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Lista de Precios | Identificador unico de una lista de precios | Busqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio (proveedor) | Busqueda | No | |
| Categoria de Producto | Categoria a la que pertenece el producto | Busqueda directa | No | |
| Producto | Producto, servicio o articulo | Busqueda | No | |
| Valido Desde | Fecha de inicio de validez | Fecha | No | Fecha actual |
| Clasificacion de Producto | Clasificacion del producto | Busqueda directa | No | |
| Clase de Producto | Clase a la que pertenece el producto | Busqueda directa | No | |
| Grupo de Producto | Grupo al que pertenece el producto | Busqueda directa | No | |
