# Almacenamiento por Proveedor

## Descripcion

Muestra el almacenamiento de productos por proveedor. Note que un producto puede tener muchos proveedores y este reporte puede mostrar muchas veces el mismo producto y almacenamiento por cada proveedor.

## Acceso

Menu: Gestion de Materiales → Reportes de Productos → Almacenamiento por Proveedor

## Parametros

| Ubicacion | Ubicacion dentro del almacen | Busqueda directa | No | |
| Producto | Producto, servicio o articulo | Busqueda | No | |
| Categoria de Producto | Categoria a la que pertenece el producto | Busqueda directa | No | |

| Grupo de Socio del Negocio | Grupo del socio del negocio | Busqueda directa | No | |
