# Inventario Analitico

## Descripcion

Inventario Analitico. Permite consultar el estado del inventario con informacion contable, mostrando cantidades y valores por producto, categoria y esquema contable.

## Acceso

Menu: Gestion de Materiales → Reportes de Productos → Inventario Analitico

## Parametros

| Parametro | Descripcion | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Organizacion | Entidad organizacional dentro del cliente | Busqueda directa | No | |
| Esquema Contable | Reglas de contabilidad | Busqueda directa | No | Esquema contable actual |
| Categoria de Producto | Categoria a la que pertenece el producto | Busqueda directa | No | |
| Producto | Producto, servicio o articulo | Busqueda | No | |
| Fecha Contable | Fecha de contabilizacion | Fecha | No | |
