# Definicion de Conteo Personalizado

## Descripcion

Define un conteo personalizado para inventario. Permite crear definiciones de conteo personalizadas que se utilizan en los procesos de inventario fisico.

## Acceso

Menu: Gestion de Materiales → Gestion de Inventario → Definicion de Conteo Personalizado

## Pestanas

### Definicion de Conteo Personalizado

Define las listas de conteo personalizadas para el inventario.

#### Campos principales

| Campo | Descripcion | Tipo | Obligatorio |
|---|---|---|---|
| Clave de Busqueda | Clave de busqueda para el registro en el formato requerido | Texto | Si |
| Nombre | Identificador alfanumerico de la entidad | Texto | Si |
| Regla de Conteo Personalizado | Clase de conteo personalizado para obtener la lista | Texto | No |
| Descripcion | Descripcion corta opcional del registro | Texto | No |
