# Reservas de Productos

## Descripcion

Muestra los movimientos de reservas de un producto.

## Acceso

Menu: Gestion de Materiales → Reservas de Productos

## Pestanas

### Reserva

Reserva de Producto. Muestra las reservas asociadas a los movimientos de inventario.

#### Campos principales

| Campo | Descripcion | Tipo | Obligatorio |
|---|---|---|---|
| Producto | Producto, servicio o articulo | Busqueda directa | Si |
| Tipo de Reserva | Tipo de reserva del producto | Lista | No |

| Ubicacion | Ubicacion dentro del almacen | Busqueda directa | No |

| Cantidad | Cantidad reservada del producto | Cantidad | No |

| Linea de Orden de Venta | Linea de la orden de venta | Busqueda directa | No |
