# Clase de Producto

## Descripcion

La ventana Clase de Producto permite definir diferentes clases o grupos de productos. La clase es una dimension de clasificacion adicional que identifica a que clase pertenece un producto dentro del catalogo. Al igual que la clasificacion, soporta jerarquias mediante la definicion de una clase padre.

## Acceso

Menu: Gestion de Materiales → Reglas de Material → Clase de Producto

## Pestanas

### Clase de Producto

Define agrupaciones unicas de productos por clase.

#### Campos

| Campo | Descripcion | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Clave de Busqueda | Clave unica de identificacion de la clase | Texto | Si |
| Nombre | Nombre de la clase de producto | Texto | Si |
| Descripcion | Descripcion opcional del registro | Texto | No |
| Activo | El registro esta activo en el sistema | Si/No | No |
| Predeterminada | Esta es la clase predeterminada | Si/No | No |
| Clase de Producto Padre | Clase padre en la jerarquia | Tabla Directa | No |
