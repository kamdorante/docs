# Organizacion de Producto

## Descripcion

El proceso Organizacion de Producto permite establecer y verificar la propiedad organizacional de los productos y sus entidades dependientes (Lista de Materiales, Sustitutos, Reabastecimiento, Compras, Informacion de Socios de Negocio, Costeo). Se puede seleccionar una Categoria de Producto o un Producto especifico para procesar.

## Acceso

Menu: Gestion de Materiales → Reglas de Material → Organizacion de Producto

## Parametros

| Parametro | Descripcion | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Organizacion | Organizacion a asignar como propietaria de los productos | Busqueda | No |
| Categoria de Producto | Categoria de productos a procesar | Busqueda | No |
| Producto | Producto especifico a procesar | Busqueda | No |
