# Clasificacion de Producto

## Descripcion

La ventana Clasificacion de Producto permite definir diferentes clasificaciones para los productos. Las clasificaciones son una dimension adicional de agrupacion de productos, complementaria a la Categoria de Producto, que permite organizar el catalogo de productos segun criterios especificos del negocio. Soporta jerarquias mediante la definicion de una clasificacion padre.

## Acceso

Menu: Gestion de Materiales → Reglas de Material → Clasificacion de Producto

## Pestanas

### Clasificacion de Producto

Define agrupaciones unicas de productos por clasificacion.

#### Campos

| Campo | Descripcion | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Clave de Busqueda | Clave unica de identificacion de la clasificacion | Texto | Si |
| Nombre | Nombre de la clasificacion de producto | Texto | Si |
| Descripcion | Descripcion opcional del registro | Texto | No |
| Activo | El registro esta activo en el sistema | Si/No | No |
| Predeterminada | Esta es la clasificacion predeterminada | Si/No | No |
| Clasificacion de Producto Padre | Clasificacion padre en la jerarquia | Tabla Directa | No |
