# Gestión de Distribución

Documentación de referencia del diccionario de aplicación para el módulo de Gestión de Distribución de Solop ERP. La planificación de recursos de distribución (DRP) es un método utilizado para planificar órdenes dentro de una cadena de suministro.

## Contenido

Este módulo incluye las herramientas para la gestión de distribución y planificación de recursos de la cadena de suministro.
