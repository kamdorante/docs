# Configuración de Distribución

## Descripción

La Configuración de Distribución es el formulario de configuración inicial del módulo de Gestión de Distribución. Permite establecer los parámetros y configuraciones necesarias para el funcionamiento del módulo DRP (Planificación de Recursos de Distribución), incluyendo la definición de reglas de distribución, tipos de documentos y parámetros organizacionales requeridos para el procesamiento de órdenes de distribución.

## Acceso

Menú: Gestión de Distribución → Configuración de Distribución
