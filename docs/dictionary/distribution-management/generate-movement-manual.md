# Generar Movimiento Manual

## Descripción

Este visor inteligente permite seleccionar las líneas de Órdenes de Distribución listas para generar sus movimientos de inventario de forma manual. Permite al usuario elegir las órdenes de distribución específicas para las cuales se generará el movimiento de inventario correspondiente.

## Acceso

Menú: Gestión de Distribución → Generar Movimiento Manual

## Campos del Browser

| Campo | Descripción | Filtro | Editable |
|-------|-------------|--------|----------|
| Orden | La orden es un documento de control que registra la transacción de distribución | No | No |
| Organización | Entidad organizacional dentro del cliente | Sí | No |
| No. Documento | Número de secuencia del documento | Sí | No |
| Tipo de Documento | Tipo de documento o reglas de procesamiento | Sí | No |
| Fecha Ordenada | Fecha en que se realizó la orden | Sí | No |
| Fecha Prometida | Fecha en que la orden fue prometida para entrega | Sí | No |
| Socio del Negocio | Identifica un socio del negocio | Sí | No |
| Almacén | Almacén de origen de la orden de distribución | Sí | No |
| Almacén Destino | Almacén destino de la orden de distribución | Sí | No |
| Estado del Documento | Estado actual del documento | Sí | No |
| No. Línea | Número de línea único para este documento | No | No |
| Producto | Producto, servicio o artículo incluido en la línea | Sí | No |
| Descripción | Descripción corta opcional del registro | No | No |
| Unidad de Medida | Unidad de medida del producto | No | No |
| Cantidad Ordenada | Cantidad ordenada en la línea | No | No |
| Cantidad Entregada | Cantidad que ya ha sido entregada | No | No |
| Cantidad en Tránsito | Cantidad actualmente en tránsito | No | No |
| Cantidad a Mover | Cantidad que se va a mover en este movimiento | No | Sí |
| Localizador | Localizador de origen en el almacén | Sí | No |
| Localizador Destino | Localizador destino en el almacén receptor | No | No |
