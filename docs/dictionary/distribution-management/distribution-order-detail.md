# Detalle de Orden de Distribución

## Descripción

El Informe de Detalle de Orden de Distribución incluye información detallada de las órdenes de distribución. Permite visualizar el estado y detalle de las órdenes, incluyendo información de productos, socios del negocio, fechas y localizadores de almacén.

## Acceso

Menú: Gestión de Distribución → Detalle de Orden de Distribución

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Tipo de Documento | Tipo de documento o reglas que determinan la secuencia y procesamiento | Tabla Directa | No |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No |
| Grupo de Socio de Negocio | El grupo de socio de negocio proporciona un método para definir valores predeterminados para socios individuales | Tabla Directa | No |
| Producto | Producto, servicio o artículo | Búsqueda | No |
| Fecha Ordenada | Fecha de la orden | Fecha | No |
| Fecha Prometida | Fecha en que la orden fue prometida | Fecha | No |
| Localizador | Localizador del almacén | Tabla Directa | No |
| Localizador Destino | Localizador destino del inventario | Tabla | No |
| Estado del Documento | Estado actual del documento | Lista | No |
| Descripción | Descripción corta opcional del registro | Texto | No |
