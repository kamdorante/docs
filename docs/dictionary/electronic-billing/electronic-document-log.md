# Bitácora de Documento Electrónico

## Descripción

Ventana para consultar el registro de actividad de los documentos electrónicos enviados y recibidos. Guarda toda la información de respuesta del proveedor fiscal, incluyendo el número de autorización, código QR, URL de descarga, hash de validación y los mensajes de estado de cada transacción electrónica.

## Acceso

Menú: Facturación Electrónica → Bitácora de Documento Electrónico

## Pestañas

### Log

Registro detallado de cada documento electrónico procesado. Almacena la trazabilidad completa del ciclo de vida del documento electrónico: identificador, serie, número de autorización, hash de verificación, código QR, URL de descarga, mensajes de respuesta del proveedor y estado.
