# Tipo de Impuesto Electrónico

## Descripción

Ventana para definir los tipos de impuesto electrónico utilizados en la facturación electrónica. Permite mapear los impuestos internos del sistema con los códigos de impuesto requeridos por el proveedor de facturación electrónica.

## Acceso

Menú: Facturación Electrónica → Tipo de Impuesto Electrónico

## Pestañas

### Tipo de Impuesto

Definición de los tipos de impuesto electrónico. Cada registro relaciona un impuesto del sistema con el código fiscal requerido por el proveedor electrónico.
