# Tipo de Transacción Fiscal

## Descripción

Ventana para definir y mantener los tipos de transacción fiscal utilizados en la facturación electrónica. Cada tipo de transacción fiscal asocia un tipo de documento del sistema con un código de documento fiscal requerido por el proveedor de facturación electrónica.

## Acceso

Menú: Facturación Electrónica → Tipo de Transacción Fiscal

## Pestañas

### Tipo de Transacción

Definición de los tipos de transacción fiscal. Almacena la relación entre los tipos de documento internos del sistema y los códigos de documento fiscal requeridos por el proveedor electrónico.
