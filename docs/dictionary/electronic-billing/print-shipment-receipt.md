# Imprimir Recibo de Entrega

## Descripción

Reporte para imprimir el recibo o comprobante de una entrega electrónica. El proceso busca primero el documento localmente (adjuntos del documento de entrega) y, si no se encuentra, lo descarga desde el proveedor fiscal externo. Permite obtener el XML o PDF autorizado del documento de entrega electrónico.

## Acceso

Menú: Facturación Electrónica → Imprimir Recibo de Entrega

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Buscar Localmente Primero | Si está activo, busca el archivo localmente antes de intentar descargarlo del proveedor fiscal (valor por defecto: Sí) | Sí/No | No |
| Entrega/Recepción | Documento de entrega o recepción cuyo comprobante electrónico se desea imprimir o descargar | Búsqueda | No |
