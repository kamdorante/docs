# Enviar Entrega al Proveedor Fiscal

## Descripción

Proceso para enviar documentos de entrega (remitos/guías de despacho) al proveedor de facturación electrónica para su validación y autorización fiscal. El proceso toma el documento de entrega seleccionado y lo transmite al emisor fiscal para obtener el comprobante electrónico correspondiente.

## Acceso

Menú: Facturación Electrónica → Enviar Entrega al Proveedor Fiscal

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Entrega/Recepción | Documento de entrega o recepción de materiales a enviar al proveedor fiscal | Búsqueda | No |
