# Tipos de Conversión para Facturación Electrónica

## Descripción

Administra los tipos de conversión de moneda para que las notas de crédito y resguardos anulados conserven la tasa de cambio del documento original, independientemente de la fecha en que se generen.

## Acceso

Menú: Facturación Electrónica → Tipo de Conversión

## Campos

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Conversión para el Día | Indica que el tipo de conversión fue creado para una fecha específica y conserva la tasa de ese día | Sí/No | No |
| Tipo de Conversión de Referencia | Tipo de conversión base que identifica la tasa específica del día, útil cuando coexisten distintos tipos en una misma fecha | Tabla Directa | No |
