# Plantilla de Correo

## Descripción

Ventana para mantener las plantillas de correo electrónico del sistema. Las plantillas pueden contener variables del usuario, socio del negocio u objeto de negocio (como Solicitud, Recordatorio, Workflow), que se resuelven dinámicamente al momento del envío.

## Acceso

Menú: Gestión de Relaciones → Plantilla de Correo

## Pestañas

### Plantilla de Correo

Define la plantilla con asunto, cuerpo del mensaje y variables reemplazables (ej. @Nombre@). Soporta sistemas multilingüe basados en el idioma del socio del negocio.

### Traducción

Versiones traducidas de la plantilla para diferentes idiomas del sistema.

### Correo de Usuario

Registro de correos enviados a usuarios utilizando esta plantilla.
