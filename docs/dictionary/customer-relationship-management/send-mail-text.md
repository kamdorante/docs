# Enviar Texto de Correo

## Descripción

Proceso para enviar correos electrónicos a los suscriptores activos de un Área de Interés o un Grupo de Socio del Negocio, utilizando una plantilla de correo y un usuario remitente seleccionado.

## Acceso

Menú: Gestión de Relaciones → Enviar Texto de Correo

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Plantilla de Correo | Plantilla de texto a utilizar para el correo | Sí |
| Área de Interés | Área de interés cuyos suscriptores activos recibirán el correo | No |
| Grupo de Socio del Negocio | Grupo de socios del negocio a quienes se enviará el correo | No |
| Usuario Remitente | Usuario desde el cual se envía el correo | No |
