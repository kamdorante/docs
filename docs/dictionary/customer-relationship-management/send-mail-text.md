# Enviar Texto de Correo

## Descripción

Proceso para enviar correos electrónicos a los suscriptores activos de un Área de Interés o un Grupo de Socio del Negocio desde un usuario seleccionado.

## Acceso

Menú: Gestión de Relaciones → Enviar Texto de Correo
