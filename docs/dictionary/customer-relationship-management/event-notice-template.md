# Plantilla de Notificación por Evento

## Descripción

Ventana para definir plantillas de notificación agrupadas por evento. Permite configurar qué plantilla de correo se envía automáticamente cuando se dispara un tipo de evento en una solicitud o proyecto.

## Acceso

Menú: Gestión de Relaciones → Plantilla de Notificación por Evento

## Pestañas

### Plantilla

Grupo de plantillas asociado a solicitudes y proyectos. Define el nombre y descripción del conjunto de notificaciones.

### Plantilla de Evento

Plantilla específica definida para un tipo de evento disparado. Asocia el evento con una plantilla de correo.

### Plantilla de Evento (Texto de Correo)

Detalle del texto del correo utilizado por la plantilla de evento, con asunto y cuerpo del mensaje.
