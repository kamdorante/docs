# Plantilla de Notificación por Evento

## Descripción

Ventana para definir las plantillas de notificación que se envían cuando ocurren eventos específicos en las solicitudes.

## Acceso

Menú: Gestión de Relaciones → Plantilla de Notificación por Evento
