# Copiar Organización

## Descripción

Proceso que permite copiar una organización existente y definir todas las reglas para la nueva organización basándose en la seleccionada. Se copian tipos de documentos, secuencias, definiciones de POS, cajas, almacenes, localizadores y otras entidades relacionadas.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Copiar Organización

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Organización | Organización de origen a copiar | No |
| Clave de Búsqueda | Clave única para la nueva organización | No |
| Nombre | Nombre de la nueva organización | No |
| RUT/NIF | Identificación fiscal de la nueva organización | No |
| Región | Región geográfica de la nueva organización | No |
| Ciudad | Ciudad de la nueva organización | No |
| Dirección 1 | Primera línea de dirección | No |
| Dirección 2 | Segunda línea de dirección | No |
| Dirección 3 | Tercera línea de dirección | No |
| Dirección 4 | Cuarta línea de dirección | No |
