# Proceso de Mantenimiento de Almacenamiento

## Descripción

Proceso de mantenimiento del almacenamiento para todo el inventario. Permite reprocesar todas las órdenes de venta por almacén y recrear todas las transacciones de inventario.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Proceso de Mantenimiento de Almacenamiento

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Organización | Organización a procesar | No |
| Almacén | Almacén a reprocesar | No |
| Categoría de Producto | Filtro por categoría de producto | No |
| Producto | Filtro por producto específico | No |
| Fecha de Transacción | Fecha de transacción de referencia | No |
| Eliminar Transacciones Incorrectas | Elimina transacciones con problemas detectados | No |
