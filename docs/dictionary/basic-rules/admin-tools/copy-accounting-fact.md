# Copiar Hechos Contables desde Otro Esquema

## Descripción

Proceso para copiar los hechos contables de un esquema contable de origen a otro, aplicando la tasa de conversión de moneda indicada.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Copiar Hechos Contables desde Otro Esquema Contable

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Esquema Contable | Esquema contable de origen | No |
| Fecha Contable | Fecha contable de los hechos a copiar | No |
| Tipo de Moneda | Tipo de conversión de moneda a aplicar | No |
| Tasa de Conversión | Tasa de conversión entre monedas | No |
| Cuenta de Conversión de Moneda | Cuenta contable para las diferencias de conversión | No |
