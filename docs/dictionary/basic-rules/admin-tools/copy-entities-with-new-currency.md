# Copiar Entidades con Nueva Moneda

## Descripción

Proceso para copiar todas las entidades relacionadas con una moneda y asignarles la nueva moneda. Es útil para re-conversión de moneda en el sistema, por ejemplo al cambiar la moneda funcional de la organización.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Copiar Entidades con Nueva Moneda

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Moneda | Moneda de origen | No |
| Moneda Destino | Moneda de destino para la conversión | No |
| Sufijo | Sufijo a agregar a los nombres de las entidades copiadas | No |
| Tipo de Moneda | Tipo de tasa de conversión a usar | No |
| Fecha de Transacción | Fecha para la conversión de moneda | No |
| Reconvertir Listas de Precios | Actualiza las listas de precios con la nueva moneda | No |
| Reconvertir Cuentas Bancarias | Actualiza las cuentas bancarias con la nueva moneda | No |
| Reconvertir Cajas | Actualiza las cajas con la nueva moneda | No |
| Reconvertir Flujos de Efectivo | Actualiza los flujos de efectivo con la nueva moneda | No |
| Reconvertir Puntos de Venta | Actualiza los POS con la nueva moneda | No |
| Reconvertir Definiciones de Comisión | Actualiza las comisiones con la nueva moneda | No |
| Reconvertir Fletes | Actualiza los fletes con la nueva moneda | No |
| Reconvertir Conceptos de Nómina | Actualiza los conceptos de nómina con la nueva moneda | No |
| Reconvertir Productos de Compra | Actualiza los productos de compra con la nueva moneda | No |
| Gestionar Operaciones de Cierre | Tipo de operación de cierre a realizar | No |
| Configurar Nueva Moneda | Establece valores por defecto para la nueva moneda | No |
