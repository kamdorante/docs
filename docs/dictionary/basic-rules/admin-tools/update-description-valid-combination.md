# Actualizar Descripción de Combinación Válida

## Descripción

Proceso para actualizar la descripción de las combinaciones válidas de cuentas contables en el sistema.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Actualizar Descripción de Combinación Válida

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Cuenta | Cuenta contable a actualizar | No |
