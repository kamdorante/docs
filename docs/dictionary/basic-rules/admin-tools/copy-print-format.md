# Copiar Formato de Impresión con Conversión

## Descripción

Proceso para copiar un formato de impresión y agregar columnas de conversión de moneda basadas en las columnas de monto existentes.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Copiar Formato de Impresión con Conversión

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Tipo de Moneda | Tipo de tasa de conversión a utilizar | No |
| Moneda | Moneda de destino para las columnas convertidas | No |
| Sufijo | Sufijo a agregar al nombre del nuevo formato | No |
