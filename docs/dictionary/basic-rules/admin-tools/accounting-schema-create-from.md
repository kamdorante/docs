# Crear Esquema Contable Desde

## Descripción

Proceso para crear un nuevo esquema contable a partir de uno existente, permitiendo definir la moneda y nombre del nuevo esquema.

## Acceso

Menú: Reglas Básicas → Herramientas de Administración → Crear Esquema Contable Desde

## Parámetros

| Parámetro | Descripción | Obligatorio |
|---|---|---|
| Esquema Contable | Esquema contable de origen | No |
| Moneda | Moneda del nuevo esquema contable | No |
| Nombre | Nombre del nuevo esquema contable | No |
| Por Defecto | Indica si el nuevo esquema será el predeterminado | No |
