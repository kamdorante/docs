# Etiquetas

## Descripción

Ventana para mantener las etiquetas (tags) del sistema. Las etiquetas permiten clasificar y categorizar registros de forma flexible, complementando los campos estándar de los documentos.

## Acceso

Menú: Reglas Básicas → Etiquetas → Etiquetas

## Pestañas

### Etiqueta

Define las etiquetas individuales del sistema. Incluye el tipo de etiqueta al que pertenece, nombre, clave de búsqueda y descripción.
