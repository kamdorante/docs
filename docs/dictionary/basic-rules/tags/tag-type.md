# Tipo de Etiqueta

## Descripción

Ventana para definir y mantener los tipos de etiquetas (tags) disponibles en el sistema. Los tipos de etiqueta agrupan las etiquetas individuales por categoría.

## Acceso

Menú: Reglas Básicas → Etiquetas → Tipo de Etiqueta

## Pestañas

### Tipo de Etiqueta

Define los tipos de etiqueta disponibles en el sistema, con su clave de búsqueda, nombre y descripción.

### Etiquetas

Etiquetas individuales asignadas a este tipo de etiqueta.
