# Configuración de Depreciación

Configuración de las reglas, métodos y convenciones de depreciación de activos fijos. Esta sección contiene las ventanas de configuración para los métodos de depreciación que se aplican a los grupos de activos y a los activos individuales.

## Contenido

Las ventanas de configuración de depreciación permiten definir:

- **Método de Depreciación**: Define el algoritmo utilizado para calcular la depreciación (línea recta, saldo decreciente, unidades de producción, entre otros).
- **Convención de Depreciación**: Establece las convenciones para el cálculo del primer y último período de depreciación.
- **Tipo de Depreciación**: Clasifica los tipos de depreciación disponibles en el sistema.
- **Tabla de Depreciación**: Define tablas con porcentajes de depreciación por período.

Estas configuraciones son asignadas a los grupos de activos y pueden ser sobreescritas a nivel de activo individual en la pestaña de Balances del Activo en la ventana Activo Fijo.
