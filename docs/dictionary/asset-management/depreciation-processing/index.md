# Procesamiento de Depreciación

Procesos para calcular y contabilizar la depreciación de activos fijos hacia el Libro Mayor.

## Reporte: Entrada de Gasto de Depreciación

Reporte utilizado para revisar las entradas de gasto de depreciación que aún no han sido contabilizadas.

### Descripción

Permite revisar los registros de depreciación pendientes de contabilización, con la posibilidad de filtrar por activo fijo, fecha contable y estado de procesamiento.

### Acceso

Menú: Gestión de Activos → Procesamiento de Depreciación → Entrada de Gasto de Depreciación

### Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Activo Fijo | Activo fijo a revisar | Búsqueda | No |
| Fecha Contable | Fecha de contabilización de la depreciación | Fecha | No |
| Vida Útil - Meses | Meses de vida útil del activo | Entero | No |
| Procesado | Estado de procesamiento del registro | Lista | No |
| Depreciable | Indica si el activo es depreciable | Lista | No |
| Depreciación Acumulada | Monto de depreciación acumulada | Monto | No |
| Gasto | Monto del gasto de depreciación | Monto | No |
| Importe Restante | Monto pendiente de depreciar | Monto | No |
