# Grupo de Activo Fijo

## Descripción

Ventana para definir grupos de activos fijos. El grupo determina las cuentas contables por defecto. Si un grupo se selecciona en la categoría del producto, los activos se crean automáticamente al entregar el producto.

## Acceso

Menú: Gestión de Activos → Grupo de Activos

## Pestañas

### Grupo de Activo

Configuración del grupo: nombre, tipo de depreciación, vida útil predeterminada y opciones de creación automática como activo fijo.

### Contabilidad

Configuración de cuentas contables para el grupo: activo fijo, depreciación acumulada, gasto de depreciación, ganancia y pérdida en disposición.
