# Enviar Retenciones

## Descripción

Permite Enviar las Retenciones Generadas via EMail. Visor inteligente que permite seleccionar retenciones completadas o cerradas para enviarlas por correo electrónico al socio de negocio correspondiente.

## Acceso

Menú: Gestión Financiera → Retenciones → Enviar Retenciones (Visor)

## Campos del Browser

| Campo | Descripción | Filtro | Editable |
|-------|-------------|--------|----------|
| Factura | Identificador del documento de factura | No | No |
| Transacción de Ventas | Indica si es una Transacción de Ventas | Sí | No |
| Socio de Negocio | Identifica un Socio de Negocio | Sí | No |
| No. del Documento | Número de secuencia del documento | No | No |
| Fecha de la Factura | Fecha impresa en la factura | Sí | No |
| Fecha Contable | Fecha contable del documento | No | No |
| Retención | Definición de retención para el Socio de Negocio | No | No |
| Tipo de Retención | Indica los tipos de retenciones de impuestos nacionales | Sí | No |
| Total | Total del documento incluyendo impuestos | No | No |
