# Enviar Retenciones

## Descripción

Permite Enviar las Retenciones Generadas via EMail. Proceso que gestiona el envío de documentos de retención a los socios de negocio correspondientes por correo electrónico.

## Acceso

Menú: Gestión Financiera → Retenciones → Enviar Retenciones

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Socio de Negocio | Identifica un Socio de Negocio destinatario | Búsqueda | No |
| Tipo de Retención | Indica los tipos de retenciones de impuestos nacionales | Tabla | No |
| Transacción de Ventas | Indica si es una Transacción de Ventas | Lista | No |
| Plantilla de Correo | Plantillas de texto para correspondencia | Tabla | No |
| Fecha del Documento | Fecha en que fue generado el documento | Fecha | No |
| Enviar Retención a Cola de Notificaciones | Indica si se debe enviar la retención a la cola de notificaciones | Sí/No | No |
