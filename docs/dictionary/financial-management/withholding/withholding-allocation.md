# Retención Generada

## Descripción

Asignación de Retenciones a Documentos por Cobrar o Pagar. Permite gestionar la asignación de retenciones generadas desde Facturas de Cuentas por Pagar o Cobrar.

## Acceso

Menú: Gestión Financiera → Retenciones → Retención Generada

## Pestañas

### Retención

Pestaña principal con los datos del documento de retención generado. Incluye tipo, fechas, montos, tasa aplicada, socio del negocio, facturas de origen y estado del documento.
