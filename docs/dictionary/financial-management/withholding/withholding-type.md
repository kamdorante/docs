# Tipo de Retención

## Descripción

Indica el Tipo de Retención de Impuesto. En la ventana Tipo de Retención se configuran los tipos de retención que se aplican a impuestos nacionales.

## Acceso

Menú: Gestión Financiera → Retenciones → Tipo de Retención

## Pestañas

### Tipo de Retención

Pestaña principal donde se definen los tipos de retención de impuesto nacionales, incluyendo tipos de documento de débito/crédito para retención y declaración, sello, firma y formato de impresión.

### Configuración de Retención

Define las funcionalidades de cada retención aplicada. Asocia el tipo de retención con su clase de cálculo, eventos disparadores, tablas de referencia y tipos de documento generados.

### Secuencia

Pestaña de secuencia de ejecución de las retenciones configuradas.
