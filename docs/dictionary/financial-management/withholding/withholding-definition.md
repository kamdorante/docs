# Definición de Retenciones

## Descripción

La ventana Definición de Retenciones determina el comportamiento de una retención de IVA, Impuesto Sobre la Renta y demás. En esta ventana podrá definir las distintas retenciones que serán aplicadas a documentos por pagar o cargadas desde documentos por cobrar.

## Acceso

Menú: Gestión Financiera → Retenciones → Definición de Retenciones

## Pestañas

### Definición de Retenciones

Pestaña principal que define el comportamiento de las retenciones. Establece el tipo de retención, socio del negocio, producto, impuesto, moneda y grupos a los que aplica.
