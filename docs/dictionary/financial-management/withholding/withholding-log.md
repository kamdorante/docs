# Log de Retención

## Descripción

Guarda un registro por cada documento procesado. Use esta información para saber si existe algún error con la configuración de retenciones.

## Acceso

Menú: Gestión Financiera → Retenciones → Log de Retención

## Pestañas

### Log

Pestaña de registro de log. Muestra el histórico de procesamiento de retenciones con la retención, configuración, documento generado, facturas relacionadas y comentarios del resultado.
