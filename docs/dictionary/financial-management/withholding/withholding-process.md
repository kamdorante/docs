# Procesar Retenciones

## Descripción

Proceso para ejecutar el cálculo y procesamiento de retenciones sobre documentos. Permite procesar retenciones de forma masiva aplicando filtros por tipo, configuración y definición de retención.

## Acceso

Menú: Gestión Financiera → Retenciones → Procesar Retenciones

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Tipo de Retención | Indica los tipos de retenciones de impuestos nacionales | Tabla | No |
| Configuración de Retención | Especifica la configuración de cada retención aplicada | Tabla | No |
| Retención | Definición de retención usada para definir la regla de retención para el Socio de Negocio | Tabla | No |
