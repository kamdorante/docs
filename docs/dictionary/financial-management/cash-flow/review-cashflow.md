# Detalle de Flujo de Efectivo

## Descripción

Reporte para revisar el detalle del flujo de efectivo con proyecciones de ingresos y egresos. Permite analizar los movimientos de efectivo consolidando facturas, órdenes, balances bancarios y registros manuales del flujo de caja.

## Acceso

Menú: Gestión de Flujo de Efectivo → Detalle de Flujo de Efectivo

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Fecha | Fecha de referencia para el análisis del flujo de efectivo | Fecha | No |
| Moneda | Moneda para el reporte | Tabla Directa | No |
| Grupo de Socio del Negocio | Grupo de socios del negocio a incluir | Tabla Directa | No |
| Socio del Negocio | Socio del negocio específico a analizar | Búsqueda | No |
| Incluir Compra y Venta | Indica que se contempla tanto compra como venta | Sí/No | No |
| Aplicar Script | Aplicar script personalizado al reporte | Sí/No | No |
| Incluir Registros de Flujo de Efectivo (manuales) | Incluir los registros manuales de flujo de caja | Sí/No | No |
| Incluir Órdenes | Incluir las órdenes de compra/venta en el análisis | Sí/No | No |
| Incluir Facturas | Incluir las facturas pendientes en el análisis | Sí/No | No |
| Incluir Cheques Diferidos | Incluir los cheques diferidos en el análisis | Sí/No | No |
