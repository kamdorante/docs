# Transacción Financiera

## Descripción

Historial de Transacción Financiera. Ventana que permite consultar y gestionar el historial de transacciones financieras asociadas a cuentas y lotes de transacciones.

## Acceso

Menú: Gestión Financiera → Inversiones y Préstamos → Transacción Financiera

## Pestañas

### Transacción

Registro de transacciones financieras individuales. Incluye lote de transacción, cuenta financiera, fecha, monto, tipo de transacción, amortización relacionada y documentos de origen (factura, pago, orden).
