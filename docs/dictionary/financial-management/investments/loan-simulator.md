# Simulador de Préstamos

## Descripción

Simulador de Préstamos. Formulario que permite simular las condiciones de un préstamo financiero, calculando cuotas, intereses y tabla de amortización antes de formalizar el acuerdo.

## Acceso

Menú: Gestión Financiera → Inversiones y Préstamos → Simulador de Préstamos
