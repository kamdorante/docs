# Lote de Transacción Financiera

## Descripción

Lote de Transacción Financiera. Ventana para gestionar lotes de transacciones financieras agrupadas, permitiendo su procesamiento y contabilización en conjunto.

## Acceso

Menú: Gestión Financiera → Inversiones y Préstamos → Lote de Transacción Financiera

## Pestañas

### Lote de Transacción

Encabezado del lote de transacciones financieras: tipo de documento, fecha, cuenta financiera, configuración funcional, monto total y estado del documento.

### Transacciones

Transacciones financieras individuales que componen el lote. Incluye cuenta financiera, fecha, monto, tipo de transacción y referencias a documentos relacionados.
