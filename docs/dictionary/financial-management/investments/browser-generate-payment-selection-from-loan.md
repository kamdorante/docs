# Generar Selección de Pago (Desde Préstamo)

## Descripción

Crea Líneas de Selección de Pago desde los Préstamos Seleccionados. Visor inteligente que permite seleccionar préstamos activos completados y generar las líneas de selección de pago correspondientes.

## Acceso

Menú: Gestión Financiera → Inversiones y Préstamos → Generar Selección de Pago (Desde Préstamo)

## Campos del Browser

| Campo | Descripción | Filtro | Editable |
|-------|-------------|--------|----------|
| Cuenta Financiera | Cuenta financiera asociada al préstamo | No | No |
| No. del Documento | Número de secuencia del documento | No | No |
| Fecha del Documento | Fecha en que fue generado el documento | No | No |
| Socio de Negocio | Identifica un Socio de Negocio | No | No |
| Producto Financiero | Producto financiero del préstamo | No | No |
| Moneda | Moneda para este registro | No | No |
| Monto de Capital | Monto de capital del préstamo | No | No |
| Monto Pagado | Monto ya pagado del préstamo | No | No |
| Monto de Capital Abierto | Monto de capital pendiente de pago | No | No |
| Monto del Pago | Monto que será incluido en la selección de pago | No | Sí |
| Monto de Interés | Monto de interés a pagar | No | No |
| Monto de Impuesto | Monto de impuesto a pagar | No | No |
