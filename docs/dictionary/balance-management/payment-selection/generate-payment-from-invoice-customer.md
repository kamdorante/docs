# Generar Pago (Desde Factura Cliente)

## Descripción
Visor inteligente que permite generar pagos/cobros a partir de facturas de clientes pendientes. Facilita la búsqueda y selección de facturas para la generación de cobros.

## Acceso
Menú: Gestión de Saldos Pendientes → Selección de Pago → Generar Pago (Desde Factura Cliente)

## Criterios de búsqueda
| Campo | Descripción | Tipo | Rango |
|---|---|---|---|
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No |
| Tipo de Documento | Tipo de documento o reglas | Búsqueda directa | No |
| Fecha de Facturación | Fecha impresa en la factura | Fecha | Sí |
| Fecha de Vencimiento | Fecha de vencimiento del cobro | Fecha | Sí |

## Columnas de resultado
| Campo | Descripción |
|---|---|
| Socio del Negocio | Nombre del socio del negocio |
| Factura | Número de documento de la factura |
| Fecha de Facturación | Fecha de la factura |
| Monto Total | Monto total de la factura |
| Monto Abierto | Monto pendiente de cobro |
| Monto del Pago | Monto a cobrar |
