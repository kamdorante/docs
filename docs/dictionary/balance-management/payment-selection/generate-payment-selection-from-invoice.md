# Generar Selección de Pago (Desde Factura)

## Descripción
Visor inteligente que permite generar una selección de pago a partir de facturas pendientes. Facilita la búsqueda y selección de facturas de proveedores para la generación de pagos.

## Acceso
Menú: Gestión de Saldos Pendientes → Selección de Pago → Generar Selección de Pago (Desde Factura)

## Criterios de búsqueda
| Campo | Descripción | Tipo | Rango |
|---|---|---|---|
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No |
| Tipo de Documento | Tipo de documento o reglas | Búsqueda directa | No |
| Fecha de Facturación | Fecha impresa en la factura | Fecha | Sí |
| Fecha de Vencimiento | Fecha de vencimiento del pago | Fecha | Sí |
| Orden | Orden de compra/venta | Búsqueda | No |

## Columnas de resultado
| Campo | Descripción |
|---|---|
| Socio del Negocio | Nombre del socio del negocio |
| Factura | Número de documento de la factura |
| Fecha de Facturación | Fecha de la factura |
| Monto Total | Monto total de la factura |
| Monto Abierto | Monto pendiente de pago |
| Monto del Pago | Monto a pagar |
