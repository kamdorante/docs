# Lote de Pagos

## Descripción
Ventana para gestionar lotes de pagos de transferencia electrónica de fondos (EFT). Permite agrupar múltiples pagos en un solo lote para procesamiento bancario.

## Acceso
Menú: Gestión de Saldos Pendientes → Selección de Pago → Lote de Pagos

## Pestañas

### Lote de Pagos
Configuración del lote de pagos.

#### Campos principales
| Campo | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| No. del Documento | Número de secuencia del documento | Texto | Sí |
| Nombre | Identificador alfanumérico de la entidad | Texto | Sí |
| Procesador de Pagos | Procesador para pagos electrónicos | Búsqueda directa | Sí |
| Fecha del Documento | Fecha del documento | Fecha | Sí |
| Monto de Procesamiento | Monto de procesamiento | Monto | No |
