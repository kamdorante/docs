# Detalle de Selección de Pago

## Descripción
Reporte que muestra el detalle de las selecciones de pago, incluyendo las facturas incluidas, montos y estados.

## Acceso
Menú: Gestión de Saldos Pendientes → Selección de Pago → Detalle de Selección de Pago

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Selección de Pago | Selección de pago a consultar | Búsqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| Cuenta Bancaria | Cuenta en el banco | Búsqueda directa | No | |
| Fecha del Documento | Fecha del documento | Fecha | No | |
