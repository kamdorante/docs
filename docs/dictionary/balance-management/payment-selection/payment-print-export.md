# Imprimir/Exportar Pagos

## Descripción
Formulario para imprimir o exportar los pagos generados a partir de una selección de pago. Permite la impresión de cheques y la exportación de archivos para transferencias electrónicas.

## Acceso
Menú: Gestión de Saldos Pendientes → Selección de Pago → Imprimir/Exportar Pagos

## Funcionalidad
Este formulario permite:
- Seleccionar una selección de pago procesada
- Imprimir cheques según el formato configurado
- Exportar archivos de pago electrónico (EFT)
- Confirmar la impresión o exportación de los pagos
- Reimprimir pagos si es necesario
