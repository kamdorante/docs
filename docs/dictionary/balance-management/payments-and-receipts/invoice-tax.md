# Impuesto de Documento por Pagar

## Descripción
Las listas del informe facturan líneas del impuesto con la información de impuesto del socio del negocio.

## Acceso
Menú: Gestión de Saldos Pendientes → Pagos y Cobros → Impuesto de Documento por Pagar

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Organización | Entidad organizacional dentro del cliente | Búsqueda directa | No | |
| Fecha de Facturación | Fecha impresa en la factura | Fecha | No | |
| Fecha Contable | Fecha contable | Fecha | No | |
| Impuesto | Identificador de impuesto | Búsqueda directa | No | |
| Tipo de Documento | Tipo de documento o reglas | Búsqueda directa | No | |
