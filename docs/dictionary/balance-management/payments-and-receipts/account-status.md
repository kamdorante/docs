# Estado de Cuenta Corriente

## Descripción
Reporte de estado de cuenta corriente que muestra los movimientos de cuenta de un socio del negocio.

## Acceso
Menú: Gestión de Saldos Pendientes → Pagos y Cobros → Estado de Cuenta Corriente

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Transacción de Ventas | Indica si es una transacción de ventas | Sí/No | No | Sí |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| Moneda | Moneda para este registro | Tabla | No | |
| Fecha Inicio | Primera fecha efectiva (inclusive) | Fecha | No | Fecha actual |
