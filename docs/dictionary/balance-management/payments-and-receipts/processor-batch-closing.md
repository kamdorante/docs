# Cierre de Lote de Procesador de Pagos

## Descripción
Cierra el lote diario del procesador de pagos. Este proceso consolida y cierra las transacciones procesadas durante el período.

## Acceso
Menú: Gestión de Saldos Pendientes → Pagos y Cobros → Cierre de Lote de Procesador de Pagos

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria | Cuenta en el banco | Búsqueda | No | |
| Procesador de Pagos | Procesador para pagos electrónicos | Búsqueda directa | No | |
| Método de Pago | Método de pago de la tienda | Búsqueda directa | No | |
| Fecha del Documento | Fecha del documento | Fecha | No | Fecha actual |
