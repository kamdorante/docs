# Detalle de Diario de Caja

## Descripción
Información de detalle de asientos de caja. Este reporte muestra información detallada de las transacciones registradas en el diario de caja.

## Acceso
Menú: Gestión de Saldos Pendientes → Diario de Caja → Detalle de Diario de Caja

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Libro de Caja | Libro de caja para registrar transacciones de caja chica | Búsqueda directa | No | |
| Organización | Entidad organizacional dentro del cliente | Búsqueda directa | No | |
| Fecha Contable | Fecha contable utilizada en las entradas del libro mayor | Fecha | No | Fecha actual |
