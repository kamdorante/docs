# Pagos sin Conciliar

## Descripción
Reporte de pagos que no han sido conciliados con el estado de cuenta bancario.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Pagos sin Conciliar

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria | Cuenta en el banco | Búsqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| Fecha de Transacción | Fecha de la transacción | Fecha | No | |
