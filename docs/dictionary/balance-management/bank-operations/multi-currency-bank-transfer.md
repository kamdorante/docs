# Transferencia Bancaria Multimoneda

## Descripción
Proceso para realizar transferencias bancarias entre cuentas de diferentes monedas, aplicando las tasas de conversión correspondientes.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Transferencia Bancaria Multimoneda

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria Desde | Cuenta bancaria origen | Búsqueda directa | Sí | |
| Cuenta Bancaria Hasta | Cuenta bancaria destino | Búsqueda directa | Sí | |
| Monto Origen | Monto en la moneda de origen | Monto | Sí | |
| Monto Destino | Monto en la moneda de destino | Monto | Sí | |
| Fecha de Transacción | Fecha de la transferencia | Fecha | Sí | Fecha actual |
| Fecha Contable | Fecha contable | Fecha | Sí | Fecha actual |
| Descripción | Descripción de la transferencia | Texto | No | |
