# Transferencia Bancaria

## Descripción
Proceso para realizar transferencias entre cuentas bancarias de la misma organización.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Transferencia Bancaria

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria Desde | Cuenta bancaria origen | Búsqueda directa | Sí | |
| Cuenta Bancaria Hasta | Cuenta bancaria destino | Búsqueda directa | Sí | |
| Socio del Negocio | Socio del negocio para la transferencia | Búsqueda | No | |
| Moneda | Moneda de la transferencia | Búsqueda directa | Sí | |
| Monto | Monto a transferir | Monto | Sí | |
| Fecha de Transacción | Fecha de la transferencia | Fecha | Sí | Fecha actual |
| Fecha Contable | Fecha contable | Fecha | Sí | Fecha actual |
| Descripción | Descripción de la transferencia | Texto | No | |
| No. de Referencia | Número de referencia de la transferencia | Texto | No | |
