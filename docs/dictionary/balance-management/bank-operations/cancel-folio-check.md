# Cancelar Folio de Cheque

## Descripción
Proceso para cancelar un folio de cheque, marcándolo como anulado en el sistema.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Cancelar Folio de Cheque

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria | Cuenta en el banco | Búsqueda directa | Sí | |
| No. de Cheque | Número de cheque a cancelar | Texto | Sí | |
