# Reporte de Estado de Cuenta Bancario

## Descripción
Reporte detallado del estado de cuenta bancario, mostrando todas las transacciones y saldos.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Reporte de Estado de Cuenta Bancario

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria | Cuenta en el banco | Búsqueda directa | No | |
| Fecha del Estado de Cuenta | Fecha del estado de cuenta | Fecha | No | |
| Estado del Documento | Estado actual del documento | Lista | No | |
