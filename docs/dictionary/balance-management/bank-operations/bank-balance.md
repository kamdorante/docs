# Saldo Bancario

## Descripción
Reporte de saldos bancarios que muestra el saldo actual de cada cuenta bancaria.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Saldo Bancario

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Cuenta Bancaria | Cuenta en el banco | Búsqueda directa | No | |
| Fecha | Fecha de consulta del saldo | Fecha | No | |
