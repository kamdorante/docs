# Conciliación de Estado de Cuenta

## Descripción
Formulario para realizar la conciliación bancaria, comparando las transacciones del estado de cuenta bancario con los pagos y cobros registrados en el sistema.

## Acceso
Menú: Gestión de Saldos Pendientes → Operaciones Bancarias → Conciliación de Estado de Cuenta

## Funcionalidad
Este formulario permite:
- Seleccionar un estado de cuenta bancario para conciliar
- Visualizar las líneas del estado de cuenta no conciliadas
- Buscar pagos coincidentes automáticamente
- Realizar la conciliación manual de transacciones
- Crear pagos desde líneas del estado de cuenta no conciliadas
- Confirmar la conciliación de las transacciones
