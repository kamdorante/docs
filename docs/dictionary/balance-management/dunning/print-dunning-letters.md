# Imprimir Carta de Morosidad

## Descripción
(Re) Imprimir carta de morosidad o enviarla como PDF anexado a un mail para el contacto del socio del negocio.

## Acceso
Menú: Gestión de Saldos Pendientes → Morosidad → Imprimir Carta de Morosidad

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Enviar PDF por Email | Enviar documentos PDF por email al socio del negocio | Sí/No | No | No |
| Plantilla de Correo | Plantillas de texto para envíos por correo | Búsqueda directa | No | |
| Corrida de Morosidad | Corrida de morosidad | Búsqueda directa | No | |
| Solo si el Socio tiene Saldo | Incluir solo si el socio del negocio tiene saldo pendiente | Sí/No | No | Sí |
| Imprimir Solo No Procesados | Imprimir solo las entradas no procesadas (no impresas) | Sí/No | No | Sí |
