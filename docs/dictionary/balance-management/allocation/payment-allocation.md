# Asignación de Pagos

## Descripción
Formulario para asignar pagos y cobros a facturas. Permite realizar la asignación manual de pagos a una o múltiples facturas, incluyendo pagos parciales, descuentos y castigos.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Asignación de Pagos

## Funcionalidad
Este formulario permite:
- Seleccionar un socio del negocio para ver sus facturas y pagos pendientes
- Asignar pagos a facturas de forma manual
- Aplicar descuentos y castigos
- Gestionar pagos parciales y sobrepagos
- Realizar asignaciones múltiples en una sola operación
