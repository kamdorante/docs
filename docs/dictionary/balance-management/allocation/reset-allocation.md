# Asignación (Restaurar)

## Descripción
Borra asignaciones para un socio del negocio o para un grupo de socio del negocio. En contraste a "Reversar" en "Consulta de Asignación", la asignación es borrada (no se guarda log), si el período está abierto.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Asignación (Restaurar)

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Grupo de Socio del Negocio | Grupo del socio del negocio | Búsqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| Fecha Contable | Fecha contable | Fecha | No | |
| Asignación | Asignación de pago | Búsqueda | No | |
| Todas las Asignaciones | Todas las asignaciones | Sí/No | No | No |
