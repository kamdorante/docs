# Asignación (Automática con Exclusión)

## Descripción
Proceso de asignación automática que permite excluir ciertos documentos del proceso de asignación. Similar a la asignación automática pero con capacidad de exclusión.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Asignación (Automática con Exclusión)

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Grupo de Socio del Negocio | Grupo del socio del negocio | Búsqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| CxP - CxC | Incluir transacciones de cuentas por cobrar y/o por pagar | Lista | No | Ambos |
| Asignar más Antiguo Primero | Asignar pagos a la factura más antigua | Sí/No | No | Sí |
| Asignar Factura Completa | Asignar la factura completa | Sí/No | No | Sí |
