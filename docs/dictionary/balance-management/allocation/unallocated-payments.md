# Pagos sin Asignar

## Descripción
Informe de pagos no asignados a las facturas. Para las facturas de pago/asignado parcialmente consulte Detalles de Pago.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Pagos sin Asignar

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| Agente de Cobro | Agente de cobro | Tabla | No | |
