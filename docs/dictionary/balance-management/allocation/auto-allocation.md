# Asignación (Automática)

## Descripción
Asignar facturas a los pagos para un socio del negocio o un grupo de socio del negocio. Usted puede restaurar asignaciones existentes primero. La asignación se basa en: (1) pagos ingresados con referencia a facturas, (2) selecciones de pago, (3) balance del socio del negocio, (4) montos coincidentes de factura y pago.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Asignación (Automática)

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Grupo de Socio del Negocio | Grupo del socio del negocio | Búsqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| CxP - CxC | Incluir transacciones de cuentas por cobrar y/o por pagar | Lista | No | Ambos |
| Asignar más Antiguo Primero | Asignar pagos a la factura más antigua | Sí/No | No | No |
