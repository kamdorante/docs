# Facturas sin Asignar

## Descripción
Informe de facturas no asignadas a los pagos. Para las facturas parcialmente pagadas/asignadas consulte Saldos Abiertos.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Facturas sin Asignar

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| Tipo de Documento | Tipo de documento o reglas | Búsqueda directa | No | |
