# Asignación de Pagos Masiva

## Descripción
Asignación masiva de facturas a los pagos. Asignar facturas a los pagos para un socio del negocio o un grupo de socio del negocio. Usted puede restaurar asignaciones existentes primero.

## Acceso
Menú: Gestión de Saldos Pendientes → Asignación → Asignación de Pagos Masiva

## Parámetros
| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Organización | Entidad organizacional dentro del cliente | Búsqueda directa | No | Organización actual |
| Grupo de Socio del Negocio | Grupo del socio del negocio | Búsqueda directa | No | |
| Socio del Negocio | Identifica un socio del negocio | Búsqueda | No | |
| CxP - CxC | Incluir transacciones de cuentas por cobrar y/o por pagar | Lista | No | Ambos |
| Asignar Factura Completa | Asignar la factura completa | Sí/No | No | Sí |
