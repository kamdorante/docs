# Asignación Masiva Factura-Recepción

## Descripción

Proceso que permite realizar la asignación masiva entre facturas de proveedor y recepciones de material.

## Acceso

Menú: Gestión de Compras → Recepción → Asignación Masiva Factura-Recepción



## Comportamiento

Asignación masiva de facturas a recepciones
