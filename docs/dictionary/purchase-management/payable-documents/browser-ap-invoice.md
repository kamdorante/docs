# Browser Facturas por Pagar

## Descripción

Navegador inteligente que permite buscar, filtrar y procesar facturas de proveedores (Cuentas por Pagar). Facilita la selección masiva y procesamiento de múltiples facturas de forma eficiente.

## Acceso

Menú: Gestión de Compras → Documentos por Pagar → Browser AP Invoice

## Campos del Browser

| Campo | Descripción | Filtro | Editable |
|-------|-------------|--------|----------|
| Nombre | Nombre del socio de negocio | Sí | No |
| Referencia | Número de referencia de la orden | Sí | No |
| Moneda | Moneda de la factura | Sí | No |
| Fecha de Orden | Fecha de la orden de compra | No | No |
| Usuario/Contacto | Agente de compras | No | No |
| Orden de Compra | Orden de compra relacionada | No | No |
| Proveedor | Socio de negocio proveedor | Sí | No |
| Total de Líneas | Suma de líneas de la factura | No | No |
| Pagado | Indica si la factura está pagada | Sí | No |
| Estado del Documento | Estado actual del documento | Sí | No |
