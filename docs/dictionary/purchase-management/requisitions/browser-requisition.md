# Crear desde Requisición

## Descripción

Crea líneas de solicitud para cotizar desde requisiciones. Navegador inteligente que permite seleccionar líneas de requisiciones de material confirmadas para crear las líneas de una solicitud para cotización (SpC).

## Acceso

Menú: Gestión de Compras → Requisiciones → Crear desde Requisición

## Criterios de búsqueda

| Campo | Descripción | Tipo | Rango |
|---|---|---|---|
| Requisición | Requisición de material a filtrar | Búsqueda directa | No |
| Fecha del Documento | Fecha del documento de requisición | Fecha | No |
| Fecha Requerida | Fecha requerida de la requisición | Fecha | No |
| Almacén | Almacén de la requisición | ID | No |
| Producto | Producto de la línea de requisición | Búsqueda directa | No |
| Usuario/Contacto | Usuario solicitante | Tabla | No |

## Columnas de resultado

| Campo | Descripción |
|---|---|
| Línea de Requisición | Identificador de la línea de requisición |
| Requisición | Documento de requisición de material |
| Fecha del Documento | Fecha del documento de requisición |
| Fecha Requerida | Fecha en que se requiere el producto |
| Almacén | Almacén de destino |
| No. de Línea | Número de línea en la requisición |
| Producto | Producto o servicio solicitado |
| Usuario/Contacto | Usuario que realizó la solicitud |
| Cantidad | Cantidad solicitada |
| Descripción | Descripción adicional de la línea |
