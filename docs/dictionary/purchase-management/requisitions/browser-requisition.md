# Consulta de Requisiciones

## Descripción

Permite buscar y filtrar requisiciones de material para su procesamiento.

## Acceso

Menú: Gestión de Compras → Requisiciones → Consulta de Requisiciones

## Criterios de búsqueda

Este navegador permite filtrar y seleccionar registros según diversos criterios.

## Columnas de resultado

Muestra los registros que coinciden con los criterios de búsqueda seleccionados.
