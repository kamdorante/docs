# Aprobación de Requisición de Material

## Descripción

Navegador inteligente que muestra las requisiciones de material planificadas (MRP) pendientes de aprobación. Permite seleccionar múltiples requisiciones para aprobarlas según criterios de almacén, planificador, fechas, producto y socio de negocio.

## Acceso

Menú: Gestión de Compras → Requisiciones → Aprobación de Requisición de Material

## Columnas de resultado

| Campo | Descripción |
|-------|-------------|
| Planificación de Requerimiento de Material | Identificador del registro MRP |
| Almacén | Almacén de la requisición |
| Planificador | Responsable de la planificación |
| No. de Documento | Número de documento de la requisición |
| Inicio Planificado | Fecha de inicio planificada |
| Fecha del Documento | Fecha del documento |
| Fecha Requerida | Fecha en que se requiere el material |
| Prioridad | Prioridad de la requisición |
| Producto | Producto solicitado |
| Unidad de Medida | Unidad de medida del producto |
| Cantidad | Cantidad requerida |
| Socio de Negocio | Proveedor sugerido |
| Descripción | Descripción adicional |
| Monto de Línea | Monto estimado de la línea |
| Fecha Inicio | Fecha de inicio |
