# Crear OC desde Líneas de Requisición

## Descripción

Navegador inteligente que permite crear órdenes de compra directamente desde líneas de requisición de material. Facilita la conversión de requisiciones en órdenes de compra seleccionando las líneas deseadas.

## Acceso

Menú: Gestión de Compras → Requisiciones → Crear OC desde Líneas de Requisición

## Columnas de resultado

| Campo | Descripción |
|-------|-------------|
| Línea de Requisición | Identificador de la línea de requisición |
| Proyecto | Proyecto asociado |
| Fase de Proyecto | Fase del proyecto |
| Tarea de Proyecto | Tarea del proyecto |
| Fecha Requerida | Fecha en que se requiere el producto |
| Producto | Producto o servicio a comprar |

| Cantidad | Cantidad a ordenar |
