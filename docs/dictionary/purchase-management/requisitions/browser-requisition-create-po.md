# Crear OC desde Líneas de Requisición

## Descripción

Permite seleccionar líneas de requisición para generar órdenes de compra directamente.

## Acceso

Menú: Gestión de Compras → Requisiciones → Crear OC desde Líneas de Requisición

## Criterios de búsqueda

Este navegador permite filtrar y seleccionar registros según diversos criterios.

## Columnas de resultado

Muestra los registros que coinciden con los criterios de búsqueda seleccionados.
