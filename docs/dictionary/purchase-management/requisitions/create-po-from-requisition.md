# Crear OC desde Requisición

## Descripción

Este proceso genera órdenes de compra a partir de las requisiciones aprobadas.

## Acceso

Menú: Gestión de Compras → Requisiciones → Crear OC desde Requisición

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio | Valor por defecto |
|---|---|---|---|---|
| Organización | Organización para la orden | Búsqueda | Sí | - |
| Almacén | Almacén de destino | Búsqueda | No | - |
| Consolidar Documentos | Consolidar en una sola OC | Sí/No | No | No |
