# SpC sin Respuesta

## Descripción

Enumera respuestas de RfQ activas (no cerradas ni procesadas) donde la respuesta no está marcada como completa. Permite identificar qué proveedores aún no han respondido a las solicitudes para cotización enviadas.

## Acceso

Menú: Gestión de Compras → Requisiciones → SpC sin Respuesta

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Tema de SpC | Filtrar por tema de solicitud para cotización | ID | No |
| Solicitud para Cotización | Filtrar por solicitud para cotización específica | Búsqueda directa | No |
| Fecha de Respuesta | Filtrar por fecha límite de respuesta | Fecha | No |
