# Respuesta SpC (Reporte)

## Descripción

Enumera respuestas de detalle de RfQs activos (no cerrados ni procesados) donde la respuesta está marcada como completa. Proporciona el detalle de todas las cotizaciones respondidas por los proveedores para facilitar la comparación y selección.

## Acceso

Menú: Gestión de Compras → Requisiciones → Respuesta SpC

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|---|---|---|---|
| Tema de SpC | Filtrar por tema de solicitud para cotización | ID | No |
| Solicitud para Cotización | Filtrar por solicitud para cotización específica | Búsqueda directa | No |
