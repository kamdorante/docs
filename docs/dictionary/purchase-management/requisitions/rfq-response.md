# Solicitud para Cotización (Con Respuesta)

## Descripción

Ventana para gestionar las respuestas de los proveedores a las solicitudes para cotización.

## Acceso

Menú: Gestión de Compras → Requisiciones → Solicitud para Cotización (Con Respuesta)

## Pestañas

### Respuesta

Información de la respuesta del proveedor incluyendo precios, plazos de entrega y ranking.

### Línea de Respuesta

Detalle de los productos/servicios cotizados con precios y condiciones.

### Cantidad de Respuesta

Precios y descuentos por cantidad ofrecidos por el proveedor.
