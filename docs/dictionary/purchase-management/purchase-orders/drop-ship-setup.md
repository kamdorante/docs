# Configuración de Entrega Directa

## Descripción

Ventana para configurar las opciones de entrega directa (drop ship) por organización. En este esquema, el proveedor envía directamente al cliente sin pasar por el almacén propio de la empresa.

## Acceso

Menú: Gestión de Compras → Órdenes de Compra → Configuración de Entrega Directa

## Pestañas

### Entrega Directa

Configuración de los parámetros de entrega directa para la organización.
