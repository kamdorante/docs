# Configuración de Entrega Directa

## Descripción

Ventana para configurar las opciones de entrega directa (drop ship) donde el proveedor envía directamente al cliente.

## Acceso

Menú: Gestión de Compras → Órdenes de Compra → Configuración de Entrega Directa
