# Lista de Precios

## Descripción

Ventana para mantener las listas de precios de productos para socios de negocio. Las listas de precios determinan la moneda y el tratamiento fiscal. Cada lista contiene tres precios por producto: Lista, Estándar y Límite. Las versiones permiten mantener listas paralelas para diferentes rangos de fechas.

## Acceso

Menú: Gestión de Compras → Órdenes de Compra → Lista de Precios

## Pestañas

### Lista de Precios

Define las listas de precios. Determina la moneda del documento y el tratamiento fiscal aplicable.

### Versión

Mantiene las versiones de la lista de precios. Se pueden calcular automáticamente a partir de la información de compra del producto y descuentos por categoría de proveedor.

### Precio de Producto

Mantiene los precios de lista, estándar y límite para cada producto en la versión de lista de precios.

### Descuento por Volumen

Define escalas de descuento por volumen aplicables a los productos de la lista.
