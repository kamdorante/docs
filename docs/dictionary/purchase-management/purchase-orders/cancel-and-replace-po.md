# Cancelar y Reemplazar OC

## Descripción

Navegador que permite seleccionar una orden de compra para cancelarla y generar una nueva orden de reemplazo.

## Acceso

Menú: Gestión de Compras → Órdenes de Compra → Cancelar y Reemplazar OC

## Criterios de búsqueda

Este navegador permite filtrar y seleccionar registros según diversos criterios.

## Columnas de resultado

Muestra los registros que coinciden con los criterios de búsqueda seleccionados.
