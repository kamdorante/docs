# Orden de Devolución (Proveedor)

## Descripción

La ventana permite crear y gestionar las autorizaciones de devolución de material al proveedor.

## Acceso

Menú: Gestión de Compras → Órdenes de Compra → Orden de Devolución (Proveedor)
