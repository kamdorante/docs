# Lote de Pólizas Contables

## Descripción

Ventana para ingresar y modificar asientos contables manuales en lote. Permite agrupar múltiples pólizas contables para su procesamiento conjunto.

## Acceso

Menú: Gestión Contable → Asientos Contables → Lote de Pólizas
