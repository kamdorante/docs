# Lote de Pólizas Contables

## Descripción

Ventana para ingresar y modificar asientos contables manuales en lote (GL Journal Batch). Permite agrupar múltiples pólizas contables para su procesamiento conjunto. Un lote puede contener varios diarios contables, cada uno con sus respectivas líneas de débito y crédito.

## Acceso

Menú: Gestión Contable → Asientos Contables → Lote de Pólizas Contables

## Pestañas

### Lote

Define los parámetros de control del lote de diarios contables: esquema contable, moneda, tipo de documento, tipo de contabilización, monto de control y estado del documento.

### Diario

Define los parámetros de control para cada diario contable individual dentro del lote, incluyendo esquema contable, moneda y tipo de conversión.

### Línea

Define las transacciones individuales de débito y crédito del diario: combinación de cuenta, montos en moneda fuente y contable, y dimensiones contables (socio, producto, proyecto, actividad, campaña).
