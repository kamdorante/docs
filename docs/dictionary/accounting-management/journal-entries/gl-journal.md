# Póliza Contable

## Descripción

Ventana para ingresar y modificar asientos contables manuales individuales. Permite crear movimientos de débito y crédito en las cuentas contables.

## Acceso

Menú: Gestión Contable → Asientos Contables → Póliza Contable
