# Conciliación de Hechos Contables (Manual)

## Descripción

Navegador inteligente para realizar la conciliación manual de hechos contables. Permite al usuario seleccionar y emparejar manualmente los débitos y créditos de una cuenta contable para marcarlos como conciliados. Es útil cuando la conciliación automática no puede emparejar los movimientos por diferencias en montos o referencias.

## Acceso

Menú: Gestión Contable → Conciliación Contable → Conciliación de Hechos Contables (Manual)

## Campos del Navegador

| Campo | Descripción | Tipo | Obligatorio |
|-------|-------------|------|-------------|
| Organización | Entidad organizacional dentro del cliente | Tabla Directa | No |
| Esquema Contable | Reglas para la contabilización | Tabla Directa | Sí |
| Cuenta | Cuenta contable a conciliar | Tabla | Sí |
| Fecha Contable | Rango de fechas contables | Fecha | No |
| Período | Período del calendario contable | Tabla Directa | No |
| Tipo de Contabilización | Tipo de monto contabilizado | Lista | No |
| Socio de Negocio | Filtro por socio de negocio | Búsqueda | No |
| Descripción | Descripción de la transacción contable | Texto | No |
| Débito Fuente | Monto de débito en moneda fuente | Monto | No |
| Crédito Fuente | Monto de crédito en moneda fuente | Monto | No |
| Débito Contabilizado | Monto de débito en moneda contable | Monto | No |
| Crédito Contabilizado | Monto de crédito en moneda contable | Monto | No |
| Conciliado | Indica si el hecho contable ha sido conciliado | Sí/No | No |
