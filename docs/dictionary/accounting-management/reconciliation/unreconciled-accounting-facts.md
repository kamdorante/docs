# Hechos Contables sin Conciliar

## Descripción

Reporte que muestra los hechos contables que aún no han sido conciliados en las cuentas configuradas para conciliación. Permite identificar los movimientos pendientes de conciliación para su seguimiento y resolución.

## Acceso

Menú: Gestión Contable → Conciliación Contable → Hechos Contables sin Conciliar

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Organización | Entidad organizacional dentro del cliente | Tabla Directa | No |
| Esquema Contable | Reglas para la contabilización | Tabla Directa | Sí |
| Cuenta | Cuenta contable a consultar | Tabla | No |
| Fecha Contable | Rango de fechas contables de los hechos sin conciliar | Fecha | No |
| Período | Período del calendario contable | Tabla Directa | No |
| Tipo de Contabilización | Tipo de monto contabilizado (Real, Presupuesto, etc.) | Lista | No |
| Socio de Negocio | Filtro por socio de negocio | Búsqueda | No |
