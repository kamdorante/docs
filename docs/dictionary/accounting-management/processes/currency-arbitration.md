# Consulta Arbitraje de Moneda BCU

## Descripción

Proceso para consultar y actualizar las tasas de arbitraje de moneda del Banco Central del Uruguay (BCU). Permite importar automáticamente las tasas de cambio publicadas por el BCU para las distintas monedas, actualizando los tipos de conversión de moneda en el sistema.

## Acceso

Menú: Gestión Contable → Procesos Contables → Consulta Arbitraje de Moneda BCU

## Parámetros

| Parámetro | Descripción | Tipo | Obligatorio |
|-----------|-------------|------|-------------|
| Fecha | Fecha para la cual se consultarán las tasas de arbitraje del BCU | Fecha | Sí |
| Tipo de Conversión de Moneda | Tipo de conversión de moneda a actualizar | Tabla Directa | No |
| Moneda | Moneda específica a consultar (si se desea filtrar) | Tabla Directa | No |
